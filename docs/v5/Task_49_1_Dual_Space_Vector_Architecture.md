# 1. Архитектурный контекст

После ревизии v4 (Tasks 48.4–48.7) зафиксированы следующие инварианты:

1. Magno, Parvo, V1 — равноправные каналы.
2. ∆V5 обладает максимальной уникальной дисперсией (~20%).
3. Общий модуль (Global Modulator) ≈ 15%.
4. Центральное поле зрения обязательно.
5. Chained subtraction как геометрическая модель отменена.
6. Дондерс сохраняется локально.
7. Пространство v5 строится как двухуровневая модель:
   - Primary Physiological Raw Space
   - Analytical Orthogonal Space
8. Медиана и MAD являются инвариантами малых выборок.
9. Фаза 2 (Ф2) — оператор нагрузки.
10. Планируется 4 канала: V1, Parvo, Magno, Koniocellular.

# 2. Уровень I — Robust Estimation Layer (Обязательный)

Этот уровень предшествует любой геометрии.

## 2.1 Малые выборки

Каждый тест содержит 36 стимулов:
- 12 Left
- 12 Center
- 12 Right

Heavy-tail и burst-режимы подтверждены Stage 6.

Следовательно:

- Среднее RT НЕ используется как базовая метрика.
- Переход к гауссовской модели запрещён.

---

## 2.2 Робастная оценка канала

Для каждого канала X и поля F ∈ {L, C, R}:

X_F = median(RT_trials_F)

Scale_X_F = MAD(RT_trials_F)


Инварианты:

- Медиана — обязательна.
- MAD — обязательна.
- Никакого логарифмирования.
- Никакого trimming.
- Никакого перехода к mean RT.

Этот слой формирует устойчивые канальные координаты.

---

# 3. Уровень II — Primary Physiological Raw Space

После робастной оценки формируется физиологический вектор:

R ∈ ℝ¹²

R =
[
V1_L, V1_C, V1_R,
P_L, P_C, P_R,
M_L, M_C, M_R,
K_L, K_C, K_R
]


Где каждое значение — медианная оценка.

Raw Space отражает физиологию, не статистику.

Никакой ортогонализации здесь не производится.

---

# 4. Локальное применение Дондерса

Для каждого канала:


Важно:

- Дондерс применяется внутри одного поля.
- Он не используется для построения глобального базиса.
- Это устраняет мультиколлинеарность v4.

---

# 5. Уровень III — Analytical Orthogonal Space

Этот уровень строится поверх робастных координат.

## 5.1 Геометрическое агрегирование (после медианы)

Для каждого канала X:

### Канальный центр

Center_X = (X_L + X_C + X_R) / 3


Это среднее применяется к уже робастно-оценённым величинам.

Не к сырым RT.

---

## 5.2 Латерализация

Lat_X_L = X_L - X_C
Lat_X_R = X_R - X_C


Центр обязателен как якорь.

Латерализация:

- ортогональна скорости,
- не агрегируется,
- не редуцируется.

---

# 6. Global Modulator (~15%)

По полной декомпозиции дисперсии:

G = α1·V1 + α2·P + α3·M + α4·K


Где коэффициенты определяются распределением объяснённой дисперсии.

G:

- не канал,
- не физиологическая переменная,
- а ковариационная системная компонента.

Он моделируется явно.

---

# 7. Оператор нагрузки (Ф2)

Фаза 2 определяется как линейный оператор:

L_λ : R_F1 → R_F2

R_F2 = R_F1 + λ·d


Где:

- d — вектор чувствительности каналов,
- λ — уровень нагрузки.

Допускается:

- ступенчатое увеличение λ,
- канально-специфическая чувствительность,
- сохранение робастной оценки в Ф2.

---

# 8. Метрики v5

## 8.1 Канальные координаты

- Center_V1
- Center_P
- Center_M
- Center_K

## 8.2 Латерализационные координаты

- Lat_X_L
- Lat_X_R

## 8.3 Нагрузочные индексы

LoadShift_X = Center_X(F2) - Center_X(F1)

## 8.4 Severity Index

Severity = || Σ^{-1/2} (x - μ) ||


где:

- μ — робастный центроид,
- Σ — ковариационная матрица,
- используется Mahalanobis-подход Stage 8.5.

---

# 9. Чёткое разделение уровней

Robust Layer:
- median
- MAD
- trial-level устойчивость

Geometric Layer:
- линейные операции над робастными координатами
- ортогонализация
- моделирование G

Смешение этих уровней запрещено.

---

# 10. Зафиксированные инварианты

1. Медиана — обязательна.
2. MAD — обязательна.
3. Среднее RT не используется.
4. Центральное поле обязательно.
5. Magno равноправен.
6. Chained subtraction отменена.
7. Дондерс — локальный.
