# I. Исправление формулы Severity

## Проблема

В документе:
docs/for_next_chat/FILES_FOR_STAGE_9D_NEXT_CHAT.md

Severity указана как:

Severity(Z) = ||Z_patient||_2 / sqrt(3)

Это не соответствует архитектуре v5.

---

## Требование

Заменить формулу на корректную:

S(Z) = Z^T Σ^{-1} Z

Формула должна быть записана в математически корректном виде.

Проверить, что нигде в каталоге /docs/for_next_chat
не осталось некорректной версии.
