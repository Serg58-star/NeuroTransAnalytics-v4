# 1. Global Modulator (G)

В Implementation Plan указано:

G = α1·V1 + α2·P + α3·M + α4·K


## Требуется уточнить:

1. Коэффициенты α:
   - фиксированы вручную?
   - или вычисляются из ковариационной матрицы?

2. Если вычисляются:
   - используется ли реальная ковариационная матрица Σ?
   - или synthetic weighting?

## Методологическое замечание:

- Если α заданы вручную — это методологически опасно.
- Если α вычисляются из текущей выборки — требуется проверка устойчивости к малому N.

Необходимо явно описать:

- источник коэффициентов,
- алгоритм их расчёта,
- зависимость от размера выборки.

# 2. Роль MAD в Analytical Space

В Walkthrough указано:

> Median independence holding.

Однако требуется уточнить роль MAD.

## Вопросы:

1. MAD используется только как scale внутри Robust Layer?
2. Или используется также для нормировки координат?

Необходимо указать, реализована ли формула:

X_F_normalized = (X_F - median_X) / MAD_X


или же Analytical Space остаётся в миллисекундах.

## Методологическое замечание:

Если координаты не стандартизируются по MAD,
то каналы с большей дисперсией могут доминировать в Analytical Space.

Этот пункт является принципиальным для корректности ортогонализации.

# 4. Phase 2 Operator (Ф2)

Текущая формула:

R_F2 = R_F1 + λ·d


## Требуется уточнить:

1. Вектор d:
   - фиксирован?
   - канал-специфичен?
   - эмпирически оценивается?

2. Пересчитывается ли Robust Layer (median и MAD) отдельно для Ф2?

## Методологическое замечание:

Если MAD не пересчитывается во второй фазе,
может произойти искусственное занижение вариабельности нагрузки.

Необходимо явно описать:

- порядок вычислений в Ф2,
- пересчёт robust-метрик,
- влияние нагрузки на scale-параметры.
