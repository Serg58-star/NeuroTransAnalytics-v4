# 1. Цель задачи

Провести формальный аудит структуры продольной динамики синтетического генератора,
используемого в Stage 9B.

Задача — определить:

1. Является ли Q2-аттрактор логическим следствием модели.
2. Или это артефакт параметризации продольного дрейфа.
3. Имеется ли эргодичность системы.
4. Существует ли механизм возврата из зоны высокой Severity.

Никаких изменений генератора.
Только диагностика.

---

# 2. Аналитические блоки аудита

## 2.1 Декомпозиция продольного дрейфа

Для каждого субъекта и временного шага:

Разложить динамику на:

Z(t+1) = Z(t) + D(t) + ε(t)

Где:
- D(t) — детерминированный компонент
- ε(t) — стохастический шум

Оценить:

- средний вектор дрейфа
- распределение нормы дрейфа
- распределение углов между D(t) и осью Severity

Проверить:
направлен ли D(t) преимущественно радиально наружу.

---

## 2.2 Анализ возвратности (Return Probability)

Для всех субъектов, попавших в Q2:

Вычислить:

P(выхода из Q2 за k шагов)

для k = 1, 2, 3, 4

Если вероятность выхода < 5%,
система не является возвратной.

---

## 2.3 Анализ насыщения Severity

Проверить:

Имеется ли верхняя граница Severity,
при которой рост замедляется (saturation).

Оценить:

E[ΔS | S]

Построить регрессию:

ΔS = f(S)

Если наклон положителен на всём диапазоне,
модель не имеет стабилизирующего механизма.

---

## 2.4 Спектральный анализ переходов

На основе эмпирической матрицы переходов:

Вычислить:

- собственные значения
- spectral gap = 1 - |λ₂|
- stationary distribution

Если spectral gap ≈ 0,
система не смешивается (non-mixing regime).

---

## 2.5 Анализ эргодичности

Проверить:

- существует ли стационарное распределение
- совпадает ли эмпирическое распределение через T шагов
  со стационарным

Если нет — динамика не эргодична.

---

# 3. Диагностические критерии

Система признаётся структурно ригидной, если выполняется ≥2 условий:

1. P_exit(Q2, k≤4) < 0.05
2. ΔS положительно коррелирует с S на всём диапазоне
3. spectral gap < 0.05
4. stationary distribution сосредоточено >70% в Q2

---

# 4. Ограничения

Строго запрещено:

- менять параметры генератора
- корректировать шум
- менять распределения
- модифицировать Z-space
- пересчитывать Severity

Только анализ текущей версии.
