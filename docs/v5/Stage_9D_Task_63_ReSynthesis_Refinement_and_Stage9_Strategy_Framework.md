### Форма зависимости — открытый параметр

Запрещено предполагать:

- **Линейность** (RT = α × Age + β): линейная модель не верифицирована.
- **Монотонность**: направление и однородность эффекта не подтверждены.
- **Аддитивность** факторов пол + возраст: взаимодействие не исключено.

## Раздел 3.2 — Уточнение Layer 3: Load Δ-Space

### Общая посылка оператора Δ

Δ-оператор (F2 − F1, F3 − F2) теоретически устраняет абсолютный базовый уровень, переходя к относительным изменениям. Это было неявным основанием для предположения о демографической нейтральности Load Δ-Space.

### Почему это предположение некорректно по умолчанию

Δ-оператор аннулирует базовый уровень только при условии, что нагрузочный эффект **аддитивен и однороден** по отношению к демографии. Если нагрузка взаимодействует с демографическими факторами, Δ сохранит демографическую сигнатуру.

Теоретически возможные взаимодействия:

- **Age × Load**: пожилые субъекты могут демонстрировать нелинейно бо́льшее нарастание RT под нагрузкой (сниженная нейронная пластичность).  
- **Sex × Load**: женщины и мужчины могут по-разному реагировать на фазовую нагрузку вследствие различий в вегетативной регуляции или стратегии скорость/точность.  
- **Нелинейная пластичность**: индивидуальное «плато» нагрузочной реакции может различаться в зависимости от демографии.

### Формальное закрепление

> **NOT_ASSUMED**: Демографическая инвариантность Δ-пространства не доказана и не предполагается по умолчанию.
>
> Статус демографической нейтральности Load Δ-Space остаётся **открытым вопросом**, требующим независимой эмпирической верификации.

| Аспект | Статус |
| :--- | :--- |
| Δ устраняет абсолютный baseline | ✅ Аддитивный случай — да |
| Δ устраняет Age × Load взаимодействие | ❌ Не доказано |
| Δ устраняет Sex × Load взаимодействие | ❌ Не доказано |
| Нелинейные эффекты пластичности | ❌ Не проверены |

## Раздел 3.3 — Стратегический выбор Stage 9: три архитектурные альтернативы

Ни одна из стратегий не выбирается в данном документе. Все три формализованы для последующего методологического решения.

---

### Стратегия A — Δ-Only Stage 9

**Описание**: Stage 9 полностью строится в пространстве Δ-переходов. Абсолютные Trait-координаты не используются.

| Аспект | Оценка |
| :--- | :--- |
| **Плюсы** | Устраняет Layer 1 проблему (базовый RT не нужен). Максимально «внутри-субъектная» модель. Соответствует субъект-центрированной клинической методике. |
| **Риски** | Не устраняет Layer 3 проблему (Age × Load может быть в Δ). Теряет информацию о Trait-уровне субъекта (диагностический контекст). |
| **Демографическая зависимость** | Снижена на уровне Baseline; не устранена на уровне Δ-взаимодействий. |
| **Соответствие клинике** | Высокое (F1→F2→F3 — штатная логика теста). |

---

### Стратегия B — Dual-Layer Model

**Описание**: Stage 9 явно разделяет два независимых аналитических контура:

- Контур I: демографически-якорный Trait Baseline (Layer 1) — с демографической поправкой.
- Контур II: инвариантный State + Load Δ (Layers 2–3) — без демографической поправки (за исключением Δ-взаимодействий, если будут обнаружены).

| Аспект | Оценка |
| :--- | :--- |
| **Плюсы** | Сохраняет диагностическую информацию о Trait-уровне. Явно разграничивает то, что известно, от того, что неизвестно. |
| **Риски** | Требует эмпирической верификации формы Layer 1 зависимости (OPEN_PARAMETER). Двухконтурная архитектура сложнее. |
| **Демографическая зависимость** | Явно управляется в Контуре I; Контур II подлежит отдельной проверке. |
| **Соответствие клинике** | Среднее (требует нормировочной базы по демографии). |

---

### Стратегия C — Parallel Contour Model

**Описание**: Амплитудный контур (абсолютные RT, Trait) и динамический контур (State, Δ) полностью разделены и анализируются параллельно, без взаимного пересечения.

| Аспект | Оценка |
| :--- | :--- |
| **Плюсы** | Максимальная методологическая чистота. Каждый контур может быть верифицирован независимо. Нет риска смешивания демографических уровней. |
| **Риски** | Высокая сложность реализации. Требует двух независимых полных аналитических пайплайнов. Клиническая интеграция двух выводов — нетривиальная задача. |
| **Демографическая зависимость** | Амплитудный контур — демографически зависим. Динамический контур — доказано инвариантен. |
| **Соответствие клинике** | Низкое / среднее (клиника оперирует единой оценкой субъекта, а не двумя параллельными). |
