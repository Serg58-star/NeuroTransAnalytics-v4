1. Где мы находимся сейчас (статус)

Проект NeuroTransAnalytics рассматривается эволюционно, а не «с нуля».
Единственный источник истины:

репозиторий GitHub;

ветка main;

документация:

docs/old_docs — предыдущий проект (обязателен к прочтению);

новая документация — уже присутствует в main.

Принято архитектурное решение:

ядро v4 — новое;

neuro_data.db используется как вспомогательный источник, не как канон.

В текущем этапе:

только зрительные тесты;

теппинг и моторная декомпозиция — исключены, будут в следующем проекте.

2. Что уже разобрано и зафиксировано (обязательно помнить)
2.1. Данные

users.xlsx / csv:

Gender: 1 = male, 0 = female;

паспортный слой корректен и обязателен.

boxbase.xlsx / csv:

агрегированные серии;

3 теста × 36 попыток;

данные уже стабилизированы онлайн, выбросы не «чистятся» постфактум.

neuro_data.db:

не мусор;

промежуточная аналитическая БД;

содержит паспорт, серии, сессии, метаданные;

не диктует модель v4.

2.2. PSI (ключевой узел)

PSI существует:

не в таблицах реакций;

в конфигурации теста;

как часть экспериментального дизайна.

Последовательности PSI:

предзаданы;

псевдослучайны;

данные о них есть (в конфигурации / метаданных).

PSI никогда не трактуется как измеренная величина.

3. Что уже прочитано из docs/old_docs

Полностью разобран документ:

README_addendum_summary.txt

Из него жёстко зафиксированы инварианты предыдущего проекта:

Проект — анализ, не тестирование.

3 теста проходят всегда:

ПЗР (V1);

color_red (ΔV4);

shift (ΔV5/MT).

36 попыток — единица анализа.

Вариабельность RT — информативна.

Сырые данные не модифицируются.

PSI — конфигурационный параметр дизайна, а не столбец данных.
