# Stage_H3.1 — F1 Cross-Model Comparison

## Исполнитель: GoAn

## Роль ChatGPT: независимая методическая проверка

---

# 1. Основание

После завершения:

* Stage_H2.2 — корректировка базиса (Baseline-Adjusted 12D),
* Stage_H2.2b — устранение повторных тестов,
* Stage_H3 — изоляция и регрессия PSI,

установлено:

* Пространственная геометрия (Eff.Rank ≈ 6.5) устойчива.
* Демографические эффекты (Sex λ₁, Q2) не являются артефактами.
* PSI — параллельный независимый домен вариации.

Следовательно, следующим логическим шагом является сравнение **наиболее чистого и протокольно-сопоставимого компонента** двух систем:

> Ф1 (Baseline / V1-путь)

---

# 2. Цель Stage_H3.1

Сравнить базовый компонент реакции (F1 / ПЗР / V1) между:

* Исторической эмпирикой
* v5 (в её базовой фазе, без Ф2 и без конио)

Ключевой вопрос:

> Совпадает ли базовая амплитудная и спектральная структура F1 между двумя архитектурами?

Это минимально зависимый от протокола уровень сравнения.

---

# 3. Формирование сопоставимых данных

## 3.1 Историческая выборка

Использовать:

* First-Visit Only (Stage_H2.2b pool, N=1482)
* Только F1 показатели
* Без Δ, без load, без PSI

## 3.2 v5 выборка

Использовать:

* Только фазу Ф1
* Исключить:

  * конио-канал
  * Ф2
  * нагрузочные переходы
* Исключить synthetic расширения

Если v5 synthetic используется —
она должна быть приведена к F1-only конфигурации.

Deliverable:

`docs/audit/H3_1_Block0_Data_Alignment.md`

# 4. Аналитические блоки

Каждый блок → отдельный .md документ.

---

## Блок 1 — Distributional Comparison

Проверить:

* Mean / Median F1
* SD / CV
* Skewness
* Kurtosis
* QQ-plot alignment (формально описать)

Сравнить:

* историка vs v5

Deliverable:

`docs/audit/H3_1_Block1_F1_Distribution.md`

Формальный вывод:

* DISTRIBUTION_ALIGNED
* DISTRIBUTION_SHIFTED
* DISTRIBUTION_INCOMPATIBLE

---

## Блок 2 — Spectral Geometry (F1 Only)

Если F1 представлена в развернутом канальном/латеральном виде:

* Построить ковариационную матрицу F1
* Вычислить:

  * Eff.Rank
  * λ-спектр
  * PC1%

Сравнить историка и v5.

Deliverable:

`docs/audit/H3_1_Block2_F1_Spectral_Geometry.md`

---

## Блок 3 — Demographic Structure in F1

Проверить:

`docs/audit/H3_1_Block3_F1_Demography.md`

## Блок 4 — Variance Decomposition (F1)
