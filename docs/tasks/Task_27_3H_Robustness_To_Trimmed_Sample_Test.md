# Task_27_3H_Robustness_To_Trimmed_Sample_Test

## Цель

Проверить устойчивость латентной структуры residual-пространства
к удалению крайних 10% субъектов.

Удаление производится не случайно,
а на основе экстремальности по residual-норме.

---

# I. Пространство анализа

Core residual-признаки:

- ΔV4_left_residual
- ΔV4_right_residual
- ΔV5_left_residual
- ΔV5_right_residual

Расширенный анализ выполнить отдельно.

---

# II. Определение "крайних субъектов"

1. Для каждого субъекта вычислить:

   Residual Norm = sqrt(sum(residual_i^2))

2. Отсортировать субъектов по этой норме.

3. Удалить:

   - верхние 10% (наиболее экстремальные)

Дополнительно выполнить симметричный тест:

   - удалить нижние 10%
   - удалить 10% случайных

---

# III. Для каждого варианта выполнить PCA

После удаления:

1. Стандартизация (z-score)
2. PCA
3. Рассчитать:

   - PC1 %
   - PC2 %
   - cumulative %
   - λ>1 count
   - Participation Ratio
   - Effective Rank

---

# IV. Метрики сравнения

Сравнить с исходной полной выборкой:

1. ΔPC1
2. ΔPC2
3. ΔParticipation Ratio
4. ΔEffective Rank
5. Δλ>1

---

# V. Критерии устойчивости

Структура считается устойчивой, если:

1. |ΔPC1| < 5%
2. |ΔPC2| < 5%
3. ΔParticipation Ratio < 0.5
4. λ>1 остаётся неизменным

---

# VI. Отчёт

Создать:

Task_27_3H_Trimmed_Sample_Stability_Report.md

Структура:

1. Исходные показатели
2. После удаления top 10%
3. После удаления bottom 10%
4. После удаления random 10%
5. Таблица сравнений
6. Summary statistics

Без интерпретаций.
Без финального вывода.

---

# VII. Ограничения

Запрещено:

- Делать визуализацию
- Делать вывод A/B/C
- Делать bootstrap
