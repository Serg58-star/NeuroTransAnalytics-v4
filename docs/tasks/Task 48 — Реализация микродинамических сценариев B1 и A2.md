# 2. Сценарий B1 — PSI Recovery Response

## 2.1 Цель

Оценить влияние длины предстимульного интервала (PSI) на последующее время реакции (RT).

Гипотеза:
RT зависит от PSI нелинейно; более длинные интервалы частично восстанавливают скорость реакции.

---

## 2.2 Методология

1. Для каждого теста (Tst1, Tst2, Tst3):
   - извлечь RT по всем 36 позициям,
   - сопоставить RT_i с PSI_i (конкретным, а не средним).

2. Построить модели:

   ### 2.2.1 Линейная модель:
   RT ~ PSI

   ### 2.2.2 Квадратичная модель:
   RT ~ PSI + PSI²

   ### 2.2.3 Стратификация:
   - по цвету,
   - по полю зрения,
   - по субъекту (mixed-effects, если возможно).

3. Проверить:
   - статистическую значимость коэффициентов,
   - величину эффекта (β),
   - R²,
   - гетероскедастичность.

4. Построить:
   - график RT от PSI,
   - оценку «времени насыщения восстановления».

---

## 2.3 Обязательные ограничения

- Не усреднять по блоку.
- Не агрегировать по тесту.
- Работать с исходными 36 позициями.
- Не использовать PCA.

# 3. Сценарий A2 — Temporal Reallocation

## 3.1 Цель

Оценить изменение RT в пределах одного блока (позиции 1 → 36).

Гипотеза:
Существует линейный или нелинейный дрейф RT внутри блока (накопление утомления).

---

## 3.2 Методология

1. Для каждого блока (каждого теста):
   - рассчитать линейный наклон:
     RT_i ~ Position_i (1–36)

2. Рассчитать:
   - индивидуальный slope,
   - распределение slopes по популяции,
   - доверительные интервалы.

3. Проверить:
   - различия между Tst1, Tst2, Tst3,
   - взаимодействие с полем зрения,
   - влияние цвета.

4. Проверить нелинейность:
   - добавить Position²,
   - проверить piecewise-модель (1–12, 13–24, 25–36).
