# Task_27_3J_Longitudinal_Trajectory_Analysis_REG_8_122

## Цель

Провести продольный анализ субъектов:

- subject_id = 8  (112 обследований)
- subject_id = 122 (26 обследований)

Задача — определить характер их движения в латентном пространстве
и оценить соотношение внутрисубъектной и межсубъектной вариабельности.

---

# I. Общие требования

1. Использовать тот же PCA-базис (core residual space),
   полученный в Task 27.3E.
2. Не пересчитывать PCA.
3. Проецировать каждое обследование в уже существующее латентное пространство.
4. Никаких кластеризаций.
5. Никаких визуализаций.
6. Только числовые показатели.

---

# II. Для каждого обследования субъекта рассчитать:

1. PC1 score
2. PC2 score
3. Residual norm
4. Межполушарную разницу:
   - ΔV4_left_residual − ΔV4_right_residual
   - ΔV5_left_residual − ΔV5_right_residual

---

# III. Внутрисубъектная вариабельность

Для каждого субъекта отдельно вычислить:

1. Mean(PC1), SD(PC1)
2. Mean(PC2), SD(PC2)
3. Общую дисперсию траектории:
   Var(PC1) + Var(PC2)
4. Среднюю дистанцию между последовательными обследованиями
5. Максимальную дистанцию

---

# IV. Межсубъектная вариабельность

По всей популяции (first-visit dataset):

1. Var_pop(PC1)
2. Var_pop(PC2)

---

# V. Рассчитать отношение:

For each subject:

Within/Between ratio for PC1  
Within/Between ratio for PC2  
Within/Between total variance  

---

# VI. Проверка сценариев

## Сценарий 1 — Стабильная точка

Если:

Within variance << Between variance  
и SD(PC1), SD(PC2) малы.

---

## Сценарий 2 — Движение вдоль одной оси

Если:

Var(PC1) >> Var(PC2)  
или наоборот.

Рассчитать:

Variance ratio = Var(PC1) / Var(PC2)

---

## Сценарий 3 — Хаотическая флуктуация

Если:

Var(PC1) ≈ Var(PC2)  
и autocorrelation низкая.

---

# VII. Дополнительные тесты

Для каждого субъекта:

1. Autocorrelation PC1(t)
2. Autocorrelation PC2(t)
3. Corr(PC1, chronological order)
4. Corr(PC2, chronological order)

---

Task_27_3J_Longitudinal_Report.md

Без нейрофизиологической интерпретации.
Только числовая оценка.

   устойчивый индивидуальный тип.
