# 1. Цель поправки

Усилить методологическую строгость микродинамического анализа (B1 и A2) путём:

1. Добавления обязательной оценки статистической мощности (Power Analysis).
2. Добавления явной проверки гетероскедастичности остатков.

Эти дополнения направлены на:

- снижение риска ложных отрицательных выводов (Type II error);
- выявление систематического роста дисперсии RT внутри блока;
- проверку устойчивости регрессионных выводов.

# 2. Дополнение к Phase 1–2 (Synthetic Validation Layer)

## 2.1 Обязательная оценка статистической мощности (Power Analysis)

Добавить подраздел:

### Power Estimation on Synthetic Data

Требуется:

1. На синтетических данных (N = 50 субъектов) оценить:
   - минимальный detectable slope для модели:
     - RT ~ PSI
     - RT ~ Position
   - при уровне значимости α = 0.05.
2. Провести симуляции с варьированием:
   - fatigue_slope,
   - recovery_factor,
   - уровня шума.
3. Рассчитать:
   - вероятность Type II error (β),
   - power = 1 − β.
4. Зафиксировать:
   - при каких величинах slope модель перестаёт быть чувствительной.
   - достаточна ли выборка N=50 для детекции реалистичных эффектов.

Отчёт должен содержать:
- таблицу detectable slope;
- график power curve;
- вывод о минимально детектируемом эффекте.

# 3. Дополнение к Phase 2–3 (Regression Validation Layer)

## 3.1 Обязательная проверка гетероскедастичности

Добавить подраздел:

### Heteroskedasticity Diagnostics

Требуется:

1. Для моделей:
   - RT ~ PSI
   - RT ~ Position
   провести проверку гетероскедастичности остатков.

2. Использовать:
   - Breusch–Pagan test (предпочтительно),
   - либо White test (если требуется).

3. При выявлении гетероскедастичности:
   - применить robust standard errors (HC3 или аналог);
   - отдельно зафиксировать влияние корректировки на p-value.

4. Зафиксировать:
   - есть ли рост дисперсии RT по мере увеличения Position;
   - есть ли зависимость дисперсии от PSI.
