# Task_27_2A_Симметричная_Регрессионная_Проверка_И_Анализ_Остатков

## Цель

Провести симметричную регрессионную проверку независимости ΔV4 и ΔV5
от базовой скорости и вариабельности
для обоих латерализованных полей зрения (left и right).

Работа выполняется до production-run.
Без визуализации.
Без кластеризации.
Только числовой анализ.

Использовать реальные данные.
Synthetic data запрещены.

---

# I. Линейная регрессия (по каждому полю отдельно)

Построить модели:

1. ΔV4_left  ~ Median_ΔV1_left
2. ΔV4_right ~ Median_ΔV1_right
3. ΔV5_left  ~ Median_ΔV1_left
4. ΔV5_right ~ Median_ΔV1_right

Для каждой модели рассчитать:

- R²
- Adjusted R²
- β-коэффициент
- p-value
- стандартную ошибку
- остатки

Сравнить:

- R²_left vs R²_right
- остаточную дисперсию left vs right

---

# II. Множественная регрессия (учёт вариабельности)

Построить модели:

1. ΔV4_left  ~ Median_ΔV1_left  + MAD_ΔV1_left
2. ΔV4_right ~ Median_ΔV1_right + MAD_ΔV1_right
3. ΔV5_left  ~ Median_ΔV1_left  + MAD_ΔV1_left
4. ΔV5_right ~ Median_ΔV1_right + MAD_ΔV1_right

Для каждой модели:

- R²
- Adjusted R²
- стандартизированные β
- p-values
- остатки

Оценить:

- уменьшается ли независимый вклад ΔV4 / ΔV5
  после учёта вариабельности.

---

# III. Анализ остаточной структуры

Для каждой модели рассчитать:

ΔV4_left_residual
ΔV4_right_residual
ΔV5_left_residual
ΔV5_right_residual

Проанализировать корреляции residual с:

- PSI_tau
- Asym_ΔV1_abs
- Asym_ΔV1_rel

Если |r| > 0.3 —
указывать наличие независимой латентной компоненты.

Сравнить:

- residual_left vs residual_right
- есть ли асимметрия в остаточной структуре.

---

# IV. Проверка нелинейности

Для каждой линейной модели проверить:

ΔV4_left  ~ Median_ΔV1_left  + (Median_ΔV1_left)^2
ΔV4_right ~ Median_ΔV1_right + (Median_ΔV1_right)^2
ΔV5_left  ~ Median_ΔV1_left  + (Median_ΔV1_left)^2
ΔV5_right ~ Median_ΔV1_right + (Median_ΔV1_right)^2

Сравнить модели по:

- AIC
- BIC

Если нелинейность улучшает модель —
зафиксировать это.

---

# V. Центр (контрольный анализ)

Провести аналогичную регрессию для center:

ΔV4_center ~ Median_ΔV1_center
ΔV5_center ~ Median_ΔV1_center

Использовать center как контроль
для сравнения степени зависимости.

---

# VI. Итоговый отчёт

Подготовить документ:

Task_27_2A_Отчёт_Симметричная_Регрессия.md

Отчёт должен содержать:

1. Таблицы коэффициентов.
2. R² и Adjusted R² по каждому полю.
3. Сравнение left vs right.
4. Корреляции residual.
5. Заключение по каждому показателю:

   - Полностью объясняется скоростью.
   - Частично объясняется.
   - Имеет независимую остаточную компоненту.
   - Демонстрирует межполушарную асимметрию.

Без интерпретации кластеров.
Без PCA.
Без визуализации.

---

# Цель этапа

Определить:

1. Являются ли ΔV4 и ΔV5
   производными от общей скорости.

2. Существует ли независимая остаточная структура.

3. Есть ли межполушарные различия
   в степени зависимости.

Только после этого переходить к production-run.

Конец документа.
