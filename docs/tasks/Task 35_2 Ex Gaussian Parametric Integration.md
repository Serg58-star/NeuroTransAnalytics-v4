# Task 35.2 — Ex-Gaussian Parametric Integration (углубление Stage 6)

## Контекст

Task 35.2 является логическим продолжением Stage 6 (Variability & Tail Geometry) и Task 35.1 (Parametric Distribution Modeling).

Stage 6 установил:

* Распределение RT не является гауссовым в строгом смысле.
* MAD встроен в Speed (strong speed–stability coupling).
* Геометрия хвостов частично ортогональна Speed.

Task 35.1 показал:

* Lognormal гипотеза отвергнута популяционно.
* Normal распределение доминирует (~67% BIC).
* Ex-Gaussian стабильно описывает ~30–40% последовательностей.

Остаётся ключевой вопрос:

> Являются ли параметры Ex-Gaussian (μ, σ, τ) структурно значимыми
> компонентами латентной архитектуры NeuroTransAnalytics-v4?

Task 35.2 завершает распределительную ветвь Stage 6.

---

# Методологические ограничения

1. Используются trial-level RT без мутации данных.
2. Параметры μ, σ, τ извлекаются только для последовательностей,
   где Ex-Gaussian модель статистически предпочтительна.
3. Никакие параметры не заменяют базовые production-признаки.
4. Анализ проводится как расширение существующей архитектуры.
5. Интерпретации формулируются операционально.

---

# Этап 1 — Извлечение параметров

Для всех последовательностей, где Ex-Gaussian признан лучшей моделью (по BIC):

Извлечь:

* μ (mean normal component)
* σ (SD normal component)
* τ (exponential tail parameter)

Дополнительно:

* проверить устойчивость оценок через bootstrap;
* оценить распределение τ по субъектам.

---

# Этап 2 — Корреляционная структура

Проверить корреляции (Spearman и Pearson) параметров μ, σ, τ с:

* Speed Axis
* Lateral Axis
* Residual Axis
* Microdynamic metrics (burst frequency, ACF)
* Tail geometry metrics (skewness, tail_to_mad_ratio)

Цель:

Определить, встроен ли τ в существующие оси
или формирует частично независимый слой.

---

# Этап 3 — PCA расширенного пространства

Сформировать пространство:

[7 macro] + [micro] + [tail metrics] + [τ]

Провести:

* PCA
* Eigenvalue spectrum
* Participation Ratio
* Bootstrap stability
* Between/Within

Цель:

Проверить, создаёт ли τ дополнительную устойчивую компоненту.

---

# Этап 4 — Подтипизация Ex-Gaussian

Проверить:

1. Существует ли кластер субъектов с высоким τ.
2. Связан ли высокий τ с:

   * повышенной micro-динамикой;
   * повышенным tail_to_mad_ratio;
   * низкой общей скоростью.
3. Стабилен ли τ при повторных тестированиях (если данные доступны).

---

# Возможные сценарии

1. τ коррелирует со Speed → tail встроен в trait.
2. τ коррелирует с micro → tail отражает state-переходы.
3. τ независим → формируется новая латентная ось.
4. Ex-Gaussian отражает смешанную популяцию подтипов.

---

# Архитектурное значение

Если τ автономен:

* появляется новый структурный компонент.
* модель RT становится: Normal + state-delays.

Если τ встроен:

* Ex-Gaussian лишь уточняет существующие слои.

---

# Критерии завершения Task 35.2

1. Извлечены устойчивые параметры μ, σ, τ.
2. Определена их позиция в латентной архитектуре.
3. Проверено появление или отсутствие новой размерности.
4. Сформулирован итог о роли экспоненциального хвоста.

---
