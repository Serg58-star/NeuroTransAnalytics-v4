# Task 24 — C3.4 Scenario A0.2: Population Structures of ΔV1

## 1. Статус задачи

Уровень: C3.4 — Scenario Engine
Сценарий: A0.2 — Population Structures of ΔV1
Версия сценария: v4.0.0
Тип задачи: Расширение Scenario Engine без изменения вычислительных слоёв C3.1–C3.3

---

## 2. Цель задачи

Реализовать сценарий A0.2, направленный на структурное описание популяционного распределения базовой компоненты ΔV1.

Сценарий не выполняет:

* кластеризацию;
* классификацию субъектов;
* нормализацию значений;
* интерпретацию результатов;
* введение новых метрик.

Сценарий формирует исключительно структурное представление популяционного распределения.

## 5. Логика сценария A0.2

### 5.1 Группировка

Финальная агрегация выполняется на уровне субъекта:

```
GROUP BY subject_id
```

### 5.2 Формируемые показатели

Для каждого субъекта рассчитываются:

* median_delta_v1_subject
* mad_delta_v1_subject
* iqr_delta_v1_subject
* n_sessions

Где:

* median_delta_v1_subject — медиана по сессиям субъекта
* mad_delta_v1_subject — MAD по сессиям субъекта
* iqr_delta_v1_subject — IQR по сессиям субъекта
* n_sessions — число валидных сессий Tst1

### 5.3 Запрещено вычислять

* percentiles популяции
* z-score
* стандартизацию
* пороговые категории
* кластерные структуры

Сценарий формирует только структурный популяционный слой.

## 6. Структура выходного артефакта

Файл:

```

```

### 6.1 Формат

* Parquet
* engine="fastparquet"
* CSV fallback запрещён

### 6.2 Поля артефакта

| Поле                    | Тип    |
| ----------------------- | ------ |
| subject_id              | int    |
| median_delta_v1_subject | float  |
| mad_delta_v1_subject    | float  |
| iqr_delta_v1_subject    | float  |
| n_sessions              | int    |
| scenario_code           | string |
| scenario_version        | string |
| etl_version             | string |
| component_algo_version  | string |
| qc_version              | string |

### 6.3 Свойства артефакта

* Immutable
* Deterministic
* Reproducible
* Versioned

Повторный запуск на той же БД должен генерировать идентичный файл.
