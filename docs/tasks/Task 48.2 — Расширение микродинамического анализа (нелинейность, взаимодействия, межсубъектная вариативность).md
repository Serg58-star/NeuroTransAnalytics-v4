# 1. Основание

В Task 48 подтверждено:

- PSI оказывает значимое влияние на RT.
- Внутриблоковое утомление присутствует избирательно (особенно в Tst2).
- Эффекты различаются по тестам.

Однако:

- протестированы только линейные модели;
- не проверено взаимодействие PSI × Position;
- не проверена нелинейность восстановления;
- не исследована межсубъектная вариативность slope;
- не проверена возможная стратификация по типам динамики.

Для проектирования v5 недопустимо оставлять эти вопросы на уровне предположений.

---

# 2. Цель Task 48.2

Численно проверить и формализовать:

1. Наличие взаимодействия PSI × Position.
2. Нелинейность восстановления (PSI).
3. Нелинейность утомления (Position).
4. Межсубъектную вариативность slope (random effects).
5. Возможную кластеризацию профилей динамики.

---

# 3. Блок 1 — Взаимодействие PSI × Position

## 3.1 Модель

RT ~ log(PSI) + Position + log(PSI) × Position

Обязательно:

- раздельно для Tst1, Tst2, Tst3;
- робастные стандартные ошибки (HC3);
- проверка гетероскедастичности.

## 3.2 Проверить

- значимость interaction term;
- изменение R² по сравнению с аддитивной моделью;
- интерпретацию знака взаимодействия.

Цель:

Определить, усиливается или ослабляется эффект восстановления при накоплении утомления.

---

# 4. Блок 2 — Нелинейность восстановления (PSI)

## 4.1 Модели

1. RT ~ log(PSI)
2. RT ~ log(PSI) + [log(PSI)]²
3. Piecewise-модель (segmented regression)
4. GAM (если оправдано и без архитектурных искажений)

## 4.2 Проверить

- saturation effect;
- наличие точки перегиба;
- plateau восстановления;
- сравнение AIC/BIC моделей.

---

# 5. Блок 3 — Нелинейность утомления (Position)

## 5.1 Модели

1. RT ~ Position
2. RT ~ Position + Position²
3. Piecewise:
   - 1–12
   - 13–24
   - 25–36

## 5.2 Проверить

- наличие ускоряющегося роста RT;
- ранний адаптационный спад;
- различия между тестами.

---

# 6. Блок 4 — Межсубъектная вариативность

## 6.1 Mixed-Effects Model

RT ~ log(PSI) + Position + (log(PSI) + Position | Subject)

Проверить:

- дисперсию random slopes;
- распределение индивидуальных slopes;
- наличие крайних профилей.

## 6.2 Построить

- гистограммы slope;
- CI;
- оценить, есть ли субпопуляции.

---

# 7. Блок 5 — Кластеризация динамических профилей

## 7.1 На основе

- индивидуального PSI slope;
- индивидуального Position slope;
- interaction coefficient.

## 7.2 Методы

- K-means (с оценкой силуэта);
- Hierarchical clustering;
- Gaussian Mixture (если оправдано).

Цель:

Проверить, существует ли скрытая структура типов динамики.
