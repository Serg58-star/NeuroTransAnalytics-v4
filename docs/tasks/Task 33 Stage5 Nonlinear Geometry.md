# Task 33 — Stage 5: Нелинейная геометрия (Intrinsic / Manifold Analysis)

## Контекст

Stage 5 завершает блок структурной верификации латентного пространства.

После Stage 1–4 установлено:

* Геометрия аддитивна и порядково-инвариантна.
* Пространство минимум трёхмерно (Dim = 3 устойчиво).
* Абсолютная скорость и латерализация — ортогональные оси.
* Канальная грубая агрегация разрушает структуру.
* Конструктивные параметры задачи не создают дополнительную размерность.

Остаётся принципиальный вопрос:

> Является ли выявленная 3-мерная структура линейным приближением
> более сложного нелинейного многообразия?

Stage 5 проверяет наличие скрытой (intrinsic) размерности,
которую PCA может недооценивать.

---

# Методологические ограничения

1. Используется тот же production-набор из 7 признаков.
2. Trial-level данные не изменяются.
3. Никакие новые агрегаты не вводятся.
4. stability.py не модифицируется.
5. Все методы применяются к стандартизированному пространству (как в baseline).

Stage 5 — это исследование геометрии, а не перестройка признаков.

---

# Блок A — Оценка внутренней размерности (Intrinsic Dimensionality)

Реализовать:

1. Participation Ratio (как контроль — уже есть).
2. MLE-based Intrinsic Dimension (Levina–Bickel estimator).
3. Two-Nearest-Neighbor estimator.
4. Correlation dimension (если вычислительно допустимо).

Сравнить:

* Dim_PCA
* Dim_intrinsic

Цель:

Проверить, превышает ли нелинейная оценка размерность PCA.

---

# Блок B — Manifold Learning (без кластеризации)

Применить:

1. Isomap
2. UMAP (без supervised-меток)
3. t-SNE (как контроль визуализации)

Для каждой проекции:

* оценить trustworthiness
* оценить continuity
* проверить, сохраняется ли топология 3D пространства

Цель:

Проверить, появляется ли скрытая структура, невидимая в PCA.

---

# Блок C — Сравнение линейной и нелинейной реконструкции

1. Рассчитать reconstruction error для:

   * PCA (k=3)
   * Isomap (k=3)
   * Autoencoder (минимальной архитектуры, без переобучения)

2. Сравнить долю объяснённой дисперсии.

Цель:

Проверить, даёт ли нелинейная модель существенный выигрыш.

---

# Блок D — Проверка на скрытые криволинейные связи

Построить:

* ΔV4 vs ΔV5 (LOESS)
* Speed vs Lateral
* Residual plots

Проверить наличие:

* систематической кривизны
* S-образных зависимостей
* кластеризации вдоль кривых

---

# Метрики к отчёту

| Method | Estimated Dim | Reconstruction Error | Trustworthiness | Continuity |
| ------ | ------------- | -------------------- | --------------- | ---------- |

Дополнительно:

* Сравнение с Dim = 3 (PCA baseline)
* Графики 2D и 3D проекций
* Bootstrap устойчивость intrinsic dimension (если реализуемо)

---

# Критерии анализа

1. Совпадает ли intrinsic dimension ≈ 3?
2. Даёт ли нелинейная модель значимый выигрыш?
3. Обнаруживаются ли скрытые криволинейные структуры?
4. Разрушается ли геометрия при нелинейной реконструкции?

---

# Возможные сценарии

1. Intrinsic Dim ≈ 3 → пространство линейно аппроксимируемо.
2. Intrinsic Dim > 3 → PCA недооценивает структуру.
3. Обнаруживается криволинейный manifold.
4. Нелинейные методы не дают выигрыша → структура действительно линейная.

Все выводы фиксируются строго операционально,
без нейрофизиологической гиперинтерпретации.

---

# Формат отчёта

1. Сводная таблица intrinsic dimension.
2. Сравнение reconstruction error.
3. Визуализация manifold-проекций.
4. Аналитический раздел:

   * есть ли скрытая нелинейная структура;
   * влияет ли это на проектирование v5 системы;
   * требуется ли усложнение модели.

---

Stage 5 считается завершённым,
если нелинейная проверка полностью выполнена
и сопоставлена с линейной PCA-моделью.
