# 1. Цель поправки

Устранить методологические уязвимости при оценке информационного вклада центрального поля зрения (Center FOV):

1. Обеспечить корректность AIC/BIC сравнений.
2. Оценить стабилизирующую роль центра.
3. Провести пересчёт статистической мощности при уменьшении числа стимулов (36 → 24).
4. Исключить ложные выводы о редундантности центра.

---

# 2. Поправки по методологии

---

## 2.1 Корректность AIC/BIC сравнения

### Проблема

Нельзя сравнивать AIC/BIC моделей, обученных на разных выборках (36 триалов vs 24 триала).

### Обязательные изменения

Добавить один из двух допустимых подходов:

### Подход A (Предпочтительный)

Использовать одинаковый набор наблюдений (L/R-only subset):

- Model A’: включает фиктивный фактор Center (если требуется как baseline dummy).
- Model B’: без учета Center.

Обе модели должны использовать идентичные индексы данных.

### Подход B (Альтернативный)

Использовать:

- K-fold cross-validation;
- Out-of-sample prediction error;

вместо прямого сравнения AIC.

Обязательно указать выбранный подход в отчёте.

---

## 2.2 Проверка стабилизирующей роли центра

Добавить обязательный анализ:

1. Изменяется ли дисперсия оценки (L−R) при удалении центра?
2. Увеличивается ли variance inflation factor?
3. Меняется ли доверительный интервал латерализационного slope?
4. Увеличивается ли нестабильность mixed-effects random slopes?

Цель:

Определить, выполняет ли центр роль стабилизирующего якоря, даже если его прямой коэффициент статистически слаб.

---

## 2.3 Пересчёт статистической мощности (Power Recalibration)

Удаление центра уменьшает количество стимулов с 36 до 24.

Добавить обязательный synthetic power recalibration:

1. Рассчитать Minimum Detectable Effect (MDE) при:
   - 36 стимулов,
   - 24 стимулов.
2. Сравнить:
   - detectable PSI slope;
   - detectable fatigue slope.
3. Зафиксировать изменение вероятности Type II error.

Если снижение мощности значимо, это должно быть отражено в итоговом вердикте.

---

## 2.4 Уточнение критериев необходимости центра

Обновить критерии:

Центр может считаться избыточным только если:

1. ΔAIC корректно рассчитан на одинаковых данных.
2. Partial R² центра статистически незначим.
3. Удаление центра не ухудшает стабильность L−R.
4. Power при 24 стимулах остаётся достаточным.
5. Mixed-effects variance не возрастает.

Если хотя бы одно условие не выполняется — центр нельзя считать редундантным.
