# Task 34.1 — Stage 5: Применение Microdynamic Architecture к реальным данным

## Контекст

Task 34.1 является продолжением Task 34 (Stage 5 — Microdynamic Architecture).

В рамках Task 34:

* реализована полная аналитическая процедура блоков XX–XXIII;
* созданы и протестированы синтетические генераторы динамики (stationary, trending, autocorrelated, bursty);
* подтверждена корректность изоляции временных структур;
* соблюдены архитектурные ограничения Framework v4.

Task 34.1 переводит валидированный алгоритмический каркас на реальные данные NeuroTransAnalytics‑v4.

Это не новое направление, а операциональное завершение Stage 5.

---

# Методологические ограничения

1. Не изменяется production‑структура 7 признаков.
2. Микродинамические показатели добавляются как параллельные параметры.
3. Никакая новая ось не вводится априори.
4. Все выводы формулируются операционально.

---

# Этап 1 — Ограниченный пилотный прогон

## Выборка

Выбрать 30–50 субъектов с:

* разным уровнем общей скорости;
* разной выраженностью латерализации;
* разными значениями PSI;
* различной стабильностью (MAD).

## Задачи

Для каждого субъекта вычислить:

* block slope (тренд по 3 блокам);
* ACF lag1–lag3;
* burst frequency (fast/slow);
* среднюю длину burst;
* permutation Δ для ACF и block trend.

## Цель

Проверить:

* воспроизводимость эффектов на реальных данных;
* распределение микро-параметров;
* отсутствие артефактов.

---

# Этап 2 — Связь микродинамики с 3D-геометрией

Проверить корреляции микро-параметров с:

* Speed Axis (PC1 baseline);
* Lateral Axis;
* Residual Axis.

Методы:

* Pearson;
* Spearman;
* Partial correlation (при необходимости).

Цель:

Определить:

* встроена ли микродинамика в существующие оси;
* или формирует независимый слой вариативности.

---

# Этап 3 — Проверка дополнительной размерности

Если микро-параметры демонстрируют:

* устойчивую межсубъектную дисперсию;
* слабую корреляцию с 3 осями;

выполнить:

* PCA в расширенном пространстве (7 + micro features);
* оценить изменение Dim (λ > 1);
* рассчитать Participation Ratio;
* сравнить Between/Within.

Цель:

Проверить появление потенциальной 4-й размерности.

---

# Критерии завершения Stage 5

Stage 5 считается полностью завершённым, если:

1. Реальные данные подтверждают валидность процедуры.
2. Определено место микродинамики в структуре (встроена или автономна).
3. Сделан вывод о наличии или отсутствии дополнительной размерности.

---

# Возможные сценарии

1. Микродинамика коррелирует со Speed → новой оси нет.
2. Burst создаёт частично независимый state-компонент.
3. Динамика индивидуальна, но не популяционна.
4. Пермутационный тест показывает критичность фиксированного порядка стимулов.

---

# Формат отчёта

1. Таблица микро-параметров по субъектам.
2. Корреляционная матрица с 3 базовыми осями.
3. PCA расширенного пространства (если применимо).
4. Итоговый аналитический вывод о роли микродинамики.

---
