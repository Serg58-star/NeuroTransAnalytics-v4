## Цель

Сформировать полный числовой отчёт по результатам production-run
(Task 27.3) на основе уже рассчитанных CSV-файлов.

Никаких автоматических интерпретаций.
Никаких выводов A/B/C.
Только строгие числовые показатели.

Финальный вывод будет сделан вручную.

---

# I. Источники данных

Использовать:

- linear_regression_results.csv
- multiple_regression_results.csv
- heteroscedasticity_tests.csv

Никаких пересчётов.
Использовать только готовые результаты.

---

# II. Структура отчёта

Сформировать документ:

Task_27_3_Числовой_Отчёт_Production_Run.md

Структура:

---

## 1. Размер выборки

- Общее количество субъектов
- Количество включённых в анализ
- Количество исключённых
- % исключений
- Количество trial

---

## 2. Линейные модели

Для каждого поля (left, right, center):

### ΔV4 ~ ΔV1

- R²
- Adjusted R²
- β
- p-value
- residual_ratio

### ΔV5 ~ ΔV1

- R²
- Adjusted R²
- β
- p-value
- residual_ratio

Отдельно вывести:

- ΔR²_left_vs_right

---

## 3. Множественные модели

Для каждого поля:

### ΔV4 ~ ΔV1 + MAD

- R²_simple
- R²_multiple
- ΔR²
- F-test p-value
- Standardized β

### ΔV5 ~ ΔV1 + MAD

- те же показатели

Отдельно вывести:

- случаи ΔR² < 0.02
- случаи ΔR² ≥ 0.05

---

## 4. Residual correlations

Для каждой модели:

- Corr(residual, PSI_tau)
- Corr(residual, Asym_abs)
- Corr(residual, Asym_rel)

Выделить:

- |r| ≥ 0.3
- |r| ≥ 0.5

---

## 5. Heteroscedasticity

Для каждой модели:

- p-value BP-test
- использовались ли robust SE
- изменялась ли значимость коэффициентов

---

## 6. τ stability

- Mean τ
- SD τ
- CV τ
- Corr(τ, PSI_count)
- Corr(τ, PSI_range)
- Corr(τ, Median_ΔV1)

---

## 7. Distribution diagnostics

Для каждого ключевого признака:

- Skewness
- Kurtosis
- % выбросов >3 SD

---
