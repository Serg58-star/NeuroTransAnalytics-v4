# Task 48.4 — Информационный вклад центрального поля зрения

# 1. Основание

В исторической архитектуре СЗР:

- 36 стимулов на тест;
- 3 поля зрения (лево / центр / право);
- каждое поле получает 12 стимулов (≈33%).

При проектировании v5:

- требуется увеличение длины серий;
- планируется фаза нагрузки (Фаза 2);
- требуется сохранение латерализационной структуры;
- требуется изучение динамики нейромедиаторной регуляции.

Центральное поле занимает 33% ресурса стимулов.

Необходимо количественно определить:

- даёт ли центр независимую информацию;
- является ли он редундантным;
- усиливает ли он устойчивость латерализационной оценки;
- можно ли перераспределить его долю без потери диагностической мощности.

---

# 2. Цель Task 48.4

Количественно оценить информационный вклад центрального поля зрения в:

1. Латерализационные метрики.
2. PSI Recovery (B1).
3. Fatigue dynamics (A2).
4. Interaction (PSI × Position).
5. Межсубъектную вариативность slope.

---

# 3. Методология

## 3.1 Сценарий A — Полная модель (L + C + R)

Базовая модель:

RT ~ Field (L/C/R) + log(PSI) + Position + взаимодействия

Рассчитать:

- R²
- AIC/BIC
- значимость Field как фактора
- величину эффекта центра относительно L/R

---

## 3.2 Сценарий B — Удаление центрального поля

Повторить анализ, исключив центральные стимулы:

RT ~ Field (L/R only) + log(PSI) + Position

Сравнить:

- изменение R²
- изменение AIC
- изменение латерализационной оси
- изменение slope утомления
- изменение PSI slope

---

## 3.3 Сценарий C — Референтная модель

Использовать центр как baseline:

L' = L − C  
R' = R − C  

Сравнить устойчивость латерализационной оценки:

- дисперсию L−R без центра;
- дисперсию L'−R' с центром.

---

## 3.4 Информационная мера

Рассчитать:

- ΔAIC между моделями;
- partial R² центра;
- вклад центра в объяснение дисперсии;
- влияние центра на mixed-effects variance;
- влияние на силу кластеризации (если применимо).

---

# 4. Критерии необходимости центра

Центральное поле признаётся необходимым, если:

1. Его удаление существенно ухудшает модель (ΔAIC > 10);
2. Его вклад в R² статистически значим;
3. Он стабилизирует латерализационную оценку;
4. Его удаление увеличивает шум или дисперсию slope.

Центральное поле признаётся редундантным, если:

1. ΔAIC несущественен;
2. Его коэффициенты незначимы;
3. Латерализация сохраняется без него;
4. Микродинамика не изменяется.

---

# 5. Дополнительные проверки

- Проверить различия по Tst1, Tst2, Tst3.
- Проверить влияние центра на нелинейность.
- Проверить влияние центра на interaction term.
- Проверить влияние центра на random slopes.
