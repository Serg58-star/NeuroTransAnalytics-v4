# Stage 9C — Task 55A
# Лёгкий скрининг коэффициента упругости κ (Lightweight Screening)

## Статус
АРХИТЕКТУРНАЯ КОРРЕКЦИЯ (Stage 1 / 3)

## Цель

Быстро отобрать 2–3 кандидата κ,
устраняющих Q2-аттрактор,
без выполнения тяжёлых спектральных и геометрических тестов.

---

# 1. Ограничения

Запрещено:

- изменять Z-space
- изменять Severity
- изменять Anchored Projection
- пересчитывать MCD
- выполнять bootstrap
- выполнять spectral analysis
- считать Silhouette

---

# 2. Вычислительные ограничения

- N = 200 субъектов
- T = 5 временных точек
- Без bootstrap
- Без eigendecomposition
- Без повторного создания классов внутри цикла

---

# 3. Модель

Использовать модифицированную формулу:

Z_{t+1} = Z_t + ε_t - κ · g(S_t) · Z_t

где:

g(S_t) = S_t / (1 + S_t)

---

# 4. Диапазон κ

Grid:

κ ∈ [0.01, 0.02, ..., 0.10]

Если ни одно значение не проходит —
расширить до 0.15 шагом 0.01.

---

# 5. Метрики отбора

Проверяются только:

1. Return_Prob_Q2 ≥ 10%
2. Saturation_Slope ∈ [-0.05, 0.02]

Если оба выполнены → κ кандидат.

Остановиться после нахождения первых 3 подходящих κ.
