# Task_27_1_Exploratory_Lab_Методическая_Коррекция_MVP

## Цель

Перед запуском Exploratory Lab устранить выявленные методические упрощения,
которые могут исказить структуру латентного пространства.

Речь идёт не о расширении модели,
а о корректировке базовых вычислений.

---

# I. Коррекция расчёта асимметрии

В текущем MVP используется формула:

    Asymmetry_ΔV1 = (right - left) / center

Это методически некорректно, так как центр не является нейтральной нормировочной величиной.

Требуется:

1. Реализовать альтернативные варианты:

    A. Абсолютная асимметрия:
       |right - left|

    B. Относительная асимметрия:
       (right - left) / ((right + left)/2)

2. Ввести обе метрики как отдельные признаки:

    - Asym_ΔV1_abs
    - Asym_ΔV1_rel

3. Не использовать центр как нормировочный параметр.

---

# II. Разделение ΔV4 и ΔV5 по полям зрения

В текущей реализации ΔV4 и ΔV5 агрегированы.

Требуется:

1. Рассчитать отдельно:

    - ΔV4_left
    - ΔV4_center
    - ΔV4_right

    - ΔV5_left
    - ΔV5_center
    - ΔV5_right

2. Рассчитать:

    - Asym_ΔV4_abs
    - Asym_ΔV5_abs

3. Не агрегировать их до завершения геометрического анализа.

---

# III. Нелинейная модель PSI

В текущей версии используется линейная регрессия RT ~ PSI.

Это упрощение.

Требуется:

1. Реализовать экспоненциальную модель восстановления:

       RT(PSI) = RT₀ + β * exp(-PSI / τ)

2. Оценить параметры:

       - τ (константа восстановления)
       - β (амплитуда)

3. Сохранить также линейную модель для сравнения.

4. Добавить новый признак:

       PSI_tau

5. Проверить устойчивость оценки τ при малом числе PSI-точек.

---

# IV. Проверка корреляционной структуры

Перед применением PCA или кластеризации:

1. Построить корреляционную матрицу всех признаков.

2. Рассчитать:

       - Pearson r
       - Spearman ρ

3. Определить:

       - Есть ли признаки с |r| > 0.8
       - Есть ли доминирующая ось общей скорости

4. Рассчитать VIF (Variance Inflation Factor) для оценки мультиколлинеарности.

5. Документировать результаты до перехода к латентному анализу.

---
