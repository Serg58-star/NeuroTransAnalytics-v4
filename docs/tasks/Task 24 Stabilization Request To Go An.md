# Task 24 — Stabilization Verification Request (Post-Implementation Audit)

Сценарий A0.2 (Population Structures of ΔV1) реализован в рамках Task 24.

## 2. Обязательные подтверждения

### 2.1 Parquet Engine

Подтвердить:

* Используется ли при сохранении артефакта A0_2.parquet явное указание:

```python
engine="fastparquet"
```

* Отсутствует ли любой CSV fallback.

Если engine не указан явно — требуется исправление.

---

### 2.2 Детерминированность порядка строк

Подтвердить:

* Выполняется ли явная сортировка результата перед сохранением:

```python
sort_values(by="subject_id")
```

* Гарантируется ли стабильный порядок строк при повторном запуске пайплайна на одной и той же базе.

Если сортировка отсутствует — требуется добавить её перед экспортом.

---

### 2.3 Источник версионных параметров

Подтвердить:

* Откуда берутся следующие поля:

  * scenario_version
  * etl_version
  * component_algo_version
  * qc_version

* Не являются ли эти значения хардкодом внутри сценария.

Требование:

Версии должны извлекаться из централизованного источника (pipeline configuration / version registry), а не задаваться вручную внутри функции run_a0_2.

## 3. Ожидаемый формат ответа

Task 24 считается полностью завершённой только после:

* подтверждения явного engine="fastparquet";
