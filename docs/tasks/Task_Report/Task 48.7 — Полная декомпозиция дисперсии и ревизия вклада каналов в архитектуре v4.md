# Task 48.7 — Полная декомпозиция дисперсии и ревизия вклада каналов в архитектуре v4
## Full Variance Partitioning & Channel Contribution Recalculation

**Версия:** v1  
**Статус:** Draft — Approval Required  
**Язык итогового документа:** Русский  

---

# 1. Основание

В результате Task 48.6 было эмпирически показано, что:

- ∆V5/MT обладает значимой независимой дисперсией,
- Partial R² ≈ 35%,
- 41.4% субъектов имеют противоположную латерализацию Parvo и Magno,
- вывод о «малоинформативности» Magno был следствием агрегированной модели.

Однако:

- текущая модель v4 не проводила полной декомпозиции объяснённой дисперсии;
- вклад Tst1 (V1), ∆V4 и ∆V5 не был разделён на:
  - уникальную долю,
  - перекрывающуюся долю,
  - общий модульный фактор.

Следовательно, требуется архитектурная ревизия.

---

# 2. Цель Task 48.7

1. Выполнить полную декомпозицию объяснённой дисперсии.
2. Разделить вклад:
   - Tst1 (V1),
   - ∆V4 (Parvo),
   - ∆V5 (Magno).
3. Определить:
   - уникальный вклад каждого канала,
   - общую пересекающуюся дисперсию,
   - долю общемодулирующего фактора.
4. Проверить гипотезу о равномерном (~1/3) распределении вклада.
5. Определить, какие документы v4 требуют пересмотра.

---

# 3. Методология (обязательная)

---

## 3.1 Базовые модели

Использовать реальную базу neuro_data.db.

Зафиксировать Outcome (Stage 9A или иной утверждённый агрегат).

Построить:

Model A:
Outcome ~ Tst1

Model B:
Outcome ~ ∆V4

Model C:
Outcome ~ ∆V5

Model D:
Outcome ~ Tst1 + ∆V4

Model E:
Outcome ~ Tst1 + ∆V5

Model F:
Outcome ~ ∆V4 + ∆V5

Model G:
Outcome ~ Tst1 + ∆V4 + ∆V5

Все модели должны использовать идентичный набор наблюдений.

---

## 3.2 Обязательная декомпозиция

Выполнить:

1. Semi-partial R² (уникальная доля каждого предиктора).
2. Commonality Analysis (разделение на:
   - уникальную дисперсию,
   - все возможные пересечения).
3. Variance Partition Diagram (Venn-диаграмма).

---

## 3.3 Проверка общемодулирующего фактора

Оценить:

- долю общей дисперсии, объясняемой совместно всеми тремя компонентами.
- проверить, можно ли интерпретировать её как «глобальный регуляторный фактор».

---

# 4. Контроль методологии

Обязательно:

- использовать robust SE (HC3),
- проверить VIF,
- исключить сравнение моделей с разным N,
- указать полную R² и Adjusted R²,
- указать AIC/BIC,
- исключить интерпретацию без численных оснований.

---

# 5. Проверка гипотезы о равномерности вклада

Чётко ответить:

1. Близок ли вклад Tst1, ∆V4 и ∆V5 к равному?
2. Является ли перекрытие следствием:
   - архитектуры,
   - общего регуляторного контура?
3. Доминирует ли какой-либо канал?
