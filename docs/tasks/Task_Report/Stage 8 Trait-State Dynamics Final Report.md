## 2. Разложение Дисперсии (Block A: Variance Decomposition)

Внутриклассная корреляция (Intraclass Correlation Coefficient, ICC) измеряет долю межсубъектной (Trait) дисперсии в общей дисперсии.

На массиве скользящих окон (N ~ 20 000 суб-состояний):

- **PC1 (Speed Axis):** ICC = **0.722**. (Высокая стабильность Черты).
- **PC2 (Lateral Axis):** ICC = **0.727**. (Высокая стабильность Черты).
- **PC3 (Residual Tone):** ICC = **0.315**. (Крайняя волатильность Состояния).
- **Global Mean ICC**: **0.588**.

**Вывод:** Базисные измерения скорости и латерализации реакции являются жесткими стабильными чертами (свойствами архитектуры), тогда как третья компонента (Residual Tone) описывает турбулентное функциональное состояние, меняющееся в ходе сессии.

## 3. Топология Траекторий и Временная Структура (Block B & C)

Анализ индивидуальных траекторий точек внутри сессии:

- **Медианная длина пути (Path Length):** 5.768 единиц.
- **Средняя амплитуда шага (Step Size):** 1.605 единиц.
- **Экспонента Хёрста (Hurst Exponent):** Нативно стабилизировалась на отметке **0.500** для всех трем осей.

**Вывод:** Движение субъекта внутри своего 3D аттрактора представляет собой абсолютно чистый **Марковский случайный процесс (Random Walk / Броуновское движение)**. Нет ни "долгосрочной памяти" ($H > 0.5$), ни эффекта жесткого осциллирующего "возврата к среднему" ($H < 0.5$). Текущий триал зависит только от предыдущего, с добавлением случайного шума.
