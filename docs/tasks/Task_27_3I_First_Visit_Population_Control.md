# Task_27_3I_First_Visit_Population_Control

## Цель

Провести полный повтор латентного анализа (Task 27 A–H)
на выборке, включающей только первое обследование
каждого субъекта.

Задача — оценить влияние повторных обследований
на популяционную латентную структуру.

Никакой агрегации.
Никакого взвешивания.
Только реальные первые посещения.

---

# I. Формирование выборки

1. Для каждого subject_id определить:
   - самое раннее обследование по дате.

2. Сформировать новый dataset:
   - 1 обследование на субъекта
   - без удаления субъектов
   - без усреднения

3. Зафиксировать:
   - итоговое N субъектов
   - сравнить с исходным N сессий
   - указать процент уменьшения выборки

---

# II. Повтор анализа

Повторить полностью:

1. Регрессионный анализ
2. Residual-анализ
3. PCA (core)
4. PCA (extended)
5. Bootstrap устойчивость
6. Split-half инвариантность

Использовать те же критерии,
те же пороги,
те же параметры.

Никаких изменений алгоритма.

---

# III. Итоговый отчёт

Создать:

Task_27_3I_First_Visit_Report.md

Структура:

1. Размер выборки
2. Регрессионные показатели
3. PCA core
4. PCA extended
5. Bootstrap результаты
6. Split-half результаты

Без интерпретаций.

---

# IV. Обязательное сравнительное приложение

Создать отдельный раздел:

Task_27_3I_Comparative_Analysis.md

В этом документе провести детальное сравнение:

Исходное исследование (все обследования)
vs
First-visit исследование

---

## Для каждого показателя сравнить:

### Регрессия

- R²
- ΔR²
- residual_ratio

### PCA core

- PC1%
- PC2%
- Cumulative%
- λ>1
- Participation Ratio
- Effective Rank

### Bootstrap

- Mean PC1%
- SD PC1%
- λ>1 frequency

### Split-half

- Mean |ΔPC1|
- cos(θ_PC1)
- cos(θ_PC2)

---

# V. Требования к сравнительному анализу

1. Указать:
   - абсолютные различия
   - относительные изменения (%)
   - направление изменений (↑ / ↓)

2. Зафиксировать:
   - какие показатели изменились >5%
   - какие остались стабильными

3. Не делать финальных выводов.
4. Не интерпретировать нейрофизиологически.
5. Только количественное сравнение.

---

# VI. Цель этапа

Определить:

1. Насколько повторные обследования
   влияют на латентную геометрию.

2. Отражает ли структура
   преимущественно индивидуальный тип
   или вносит вклад состояние.

---

# VII. Ограничения

Запрещено:

- Делать кластеризацию
- Делать визуализацию
- Делать итоговый вывод A/B/C
- Делать теоретические интерпретации

---

# Итог

Этот этап должен дать строгое количественное понимание
величины и направленности изменений
между двумя вариантами анализа.

Только после этого возможен окончательный научный вывод.

Конец документа.
