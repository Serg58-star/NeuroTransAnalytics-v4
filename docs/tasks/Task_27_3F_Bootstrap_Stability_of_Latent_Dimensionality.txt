# Task_27_3F_Bootstrap_Stability_of_Latent_Dimensionality

## Цель

Проверить устойчивость латентной размерности residual-пространства
с помощью bootstrap-ресэмплинга субъектов.

Мы не оцениваем кластеры.
Мы проверяем стабильность:

- доли объяснённой дисперсии PC1, PC2
- числа λ > 1
- Participation Ratio
- Effective Rank

---

# I. Пространство анализа

Использовать core residual-признаки:

- ΔV4_left_residual
- ΔV4_right_residual
- ΔV5_left_residual
- ΔV5_right_residual

Анализ расширенного набора (7 переменных)
выполнить отдельно.

---

# II. Bootstrap процедура

1. Число итераций: 1000
2. Ресэмплинг по субъектам (с возвращением)
3. Размер каждой подвыборки = N (полная выборка)
4. На каждой итерации:
   - стандартизация (z-score)
   - PCA
   - вычисление:
       • PC1 %
       • PC2 %
       • cumulative %
       • λ > 1 count
       • Participation Ratio
       • Effective Rank

---

# III. Сохраняемые показатели

Для каждой метрики сохранить:

- Mean
- SD
- 95% CI
- Min / Max

Отдельно сохранить распределение PC1%.

---

# IV. Критерии устойчивости

Структура считается устойчивой, если:

1. SD(PC1%) < 5%
2. λ > 1 count стабилен (одинаков в ≥90% итераций)
3. Participation Ratio SD < 0.3
4. Effective Rank SD < 0.5

Не делать интерпретаций.
Только числовая оценка стабильности.

---

# V. Отчёт

Создать:

Task_27_3F_Bootstrap_Stability_Report.md

Без выводов.
Без интерпретаций.
Без кластеров.
