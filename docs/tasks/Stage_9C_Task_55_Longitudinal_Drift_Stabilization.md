# 2. Архитектурная цель

Ввести mean-reverting эластичность,
обеспечивающую:

1. Насыщение Severity при высоких значениях.
2. Ненулевую вероятность возврата из Q2.
3. Эргодичность системы.
4. Сохранение непрерывности поля (Silhouette < 0.20).

---

# 3. Математическая модель

Текущая модель (упрощённо):

Z_{t+1} = Z_t + ε_t

Новая модель:

Z_{t+1} = Z_t
          + ε_t
          - κ · g(S_t) · Z_t

где:

κ > 0 — коэффициент упругости  
g(S_t) — функция усиления при высоком Severity

Предлагаемая форма:

g(S_t) = S_t / (1 + S_t)

Свойства:

- при S → 0 → g ≈ 0 (центр стабилен)
- при S ↑ → g → 1 (усиливается возврат)
- асимптотическое ограничение дрейфа

Это дискретный аналог процесса Орнштейна–Уленбека,
но без нарушения Z-инвариантов.

---

# 4. Требования к параметру κ

Подобрать κ так, чтобы выполнялось:

1. Return_Prob_Q2 ≥ 10%
2. Saturation_Slope ≤ 0
3. Stationary Q2 ≤ 50%
4. Spectral Gap ≥ 0.1
5. Silhouette longitudinal < 0.20

Подбор осуществляется через grid-search.
