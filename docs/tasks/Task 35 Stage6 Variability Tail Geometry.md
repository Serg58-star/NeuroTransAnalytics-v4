# Task 35 — Stage 6: Variability & Tail Geometry (XXIV–XXVI)

## Контекст

Task 35 реализует **ЭТАП 6 Exploratory Architecture Framework v4**.

После завершения:

* Structural Verification (геометрия trait-пространства),
* Stage 5 — Microdynamic Architecture (state-динамика внутри теста),

возникает следующий фундаментальный вопрос:

> Формирует ли вариативность распределений RT самостоятельную латентную размерность,
> или она встроена в существующие trait- и state-компоненты?

Stage 6 анализирует форму распределений и поведение хвостов, а не временную динамику.

---

# Методологические ограничения

1. Используются trial-level RT без изменения генеративной логики.
2. Production 7-мерное пространство не модифицируется.
3. Variability-параметры добавляются как параллельные признаки.
4. Никакие новые агрегаты не заменяют базовые метрики.
5. Все выводы фиксируются операционально.

---

# XXIV. Skew / Kurtosis анализ

Для каждого субъекта и каждого теста вычислить:

* Skewness распределения RT;
* Kurtosis (excess kurtosis);
* Shapiro–Wilk или D’Agostino тест нормальности (по необходимости);
* Сравнение с симметричным bootstrap-распределением.

Цели:

1. Проверить симметричность распределений.
2. Выявить heavy-tail эффекты.
3. Оценить межсубъектную вариативность формы распределения.

Дополнительно:

* Корреляция skew/kurtosis с Speed, Lateral, Residual.
* Проверка связи skew с burst-поведением.

---

# XXV. Tail-layer (P90 − Median)

Определить Tail Index как:

Tail = P90 − Median

(при необходимости дополнительно P95 − Median)

Для каждого субъекта вычислить:

* абсолютное значение Tail;
* нормированное Tail (Tail / Median);
* отношение Tail к MAD.

Анализ:

1. Межсубъектная дисперсия Tail.
2. Корреляция Tail с Speed.
3. Корреляция Tail с Microdynamic-параметрами.
4. PCA хвостовых параметров отдельно.

Цель:

Проверить, создаёт ли tail-layer самостоятельную ось.

---

# XXVI. MAD как отдельная ось

Для каждого субъекта вычислить:

* MAD (уже существует);
* Coefficient of Variation (CV);
* Robust CV (MAD / Median).

Проверить:

1. Speed–Stability trade-off (корреляция Speed ↔ MAD);
2. Частичную корреляцию с учётом micro-динамики;
3. Включение MAD в расширенную PCA (7 + micro + variability);
4. Between/Within для variability-параметров.

Цель:

Определить, является ли стабильность независимой размерностью.

---

# Интегральный анализ Stage 6

1. Сформировать расширенное пространство:

   7 (macro) + micro (Stage 5) + variability (Stage 6)

2. Провести PCA:

   * Eigenvalues;
   * Participation Ratio;
   * Bootstrap устойчивость;
   * Split-half;

3. Проверить появление новой устойчивой компоненты.

---

# Критерии интерпретации

1. Если skew/kurtosis распределены случайно → вариативность вторична.
2. Если Tail коррелирует со Speed → вариативность встроена в trait.
3. Если MAD формирует отдельную компоненту → появляется ось стабильности.
4. Если variability независима от trait и state → появляется новая размерность.

---

# Возможные сценарии

1. Вариативность полностью встроена в Speed.
2. Speed–Stability trade-off формирует 4-ю ось.
3. Tail-layer связан с burst-динамикой.
4. Heavy tails отражают редкие state-переходы.

---

# Формат отчёта

1. Таблица skew/kurtosis по субъектам.
2. Таблица tail-параметров.
3. Корреляционная матрица macro + micro + variability.
4. PCA расширенного пространства.
5. Итоговый вывод:

   * существует ли Variability Axis;
   * как она соотносится с trait/state;
   * влияние на проектирование v5.

---

Stage 6 считается завершённым,
если установлено, формирует ли вариативность самостоятельную латентную размерность.
