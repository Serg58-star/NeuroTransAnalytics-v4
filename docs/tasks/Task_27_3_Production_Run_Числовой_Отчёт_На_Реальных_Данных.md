# Task_27_3_Production_Run_Числовой_Отчёт_На_Реальных_Данных

## Цель

Выполнить полный симметричный регрессионный анализ
на всём доступном реальном массиве данных
с использованием стабилизированного модуля symmetric_regression.py.

Этот запуск является первым окончательным числовым тестом
структурной независимости ΔV4 и ΔV5
от базовой скорости ΔV1 и вариабельности.

Никаких synthetic data.
Никаких тестовых выборок.
Использовать полный массив.

---

# I. Общие ограничения

Запрещено:

- Выполнять PCA
- Выполнять кластеризацию
- Выполнять UMAP
- Выполнять визуализацию
- Строить графики
- Делать интерпретационные выводы сверх числовых показателей

Разрешено:

- Только числовой отчёт
- Только таблицы
- Только формальные критерии

---

# II. Объём данных

В отчёте обязательно указать:

1. Общее количество субъектов в базе.
2. Количество субъектов, включённых в анализ.
3. Количество субъектов, исключённых:
   - по PSI range
   - по нестабильности τ
   - по отсутствию данных
4. Количество наблюдений по каждому полю (left/right/center).

---

# III. Линейные модели

Для каждого поля (left, right, center) выполнить:

ΔV4_field ~ Median_ΔV1_field  
ΔV5_field ~ Median_ΔV1_field  

Вывести:

- R²
- Adjusted R²
- β
- p-value
- residual_ratio

Сравнить:

- R²_left vs R²_right
- ΔR²_left_right

---

# IV. Множественные модели

Для каждого поля:

ΔV4_field ~ Median_ΔV1_field + MAD_ΔV1_field  
ΔV5_field ~ Median_ΔV1_field + MAD_ΔV1_field  

Вывести:

- R² simple
- R² multiple
- ΔR²
- F-test p-value
- Standardized β
- Robust SE (если применимо)

Отдельно указать:

- случаи ΔR² < 0.02
- случаи ΔR² ≥ 0.05

---

# V. Остаточная структура

Для каждой модели вывести:

Корреляции residual с:

- PSI_tau
- Asym_ΔV1_abs
- Asym_ΔV1_rel

Отметить случаи:

|r| ≥ 0.3  
|r| ≥ 0.5  

---

# VI. Гетероскедастичность

Для каждой модели указать:

- p-value Breusch–Pagan
- применялись ли robust SE
- изменяется ли значимость коэффициентов при HC3

---

# VII. Устойчивость τ

Вывести:

- Среднее τ
- Стандартное отклонение τ
- CV τ
- Корреляция τ с:
    - количеством PSI-наблюдений
    - range PSI
    - Median_ΔV1

Указать:

- процент исключённых субъектов
- влияет ли исключение на распределение τ

---

# VIII. Проверка распределений

Для каждого ключевого признака:

- Skewness
- Kurtosis
- % выбросов (>3 SD)

Указать признаки с тяжёлыми хвостами.

---

# IX. Анализ доминирующей оси скорости

Без выполнения PCA:

1. Рассчитать среднюю корреляцию между:
   - ΔV4_field и Median_ΔV1_field
   - ΔV5_field и Median_ΔV1_field
   - Median_ΔV1_left и Median_ΔV1_right

2. Оценить наличие доминирующей "общей скорости".

---

# X. Итоговый формальный вывод

В конце отчёта должен быть один из трёх формальных выводов:

A. ΔV4 и ΔV5 преимущественно объясняются базовой скоростью.  
B. Существует умеренная независимая остаточная структура.  
C. Выраженная независимая латентная структура присутствует.

Без философских комментариев.
Без интерпретаций.
Без предположений о диагнозах.

Только формальный вывод на основании числовых критериев.
