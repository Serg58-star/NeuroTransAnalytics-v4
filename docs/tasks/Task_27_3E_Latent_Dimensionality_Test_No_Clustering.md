# Task_27_3E_Latent_Dimensionality_Test_No_Clustering

## Цель

Определить латентную размерность пространства
после регрессионного контроля ΔV1 и MAD,
используя анализ главных компонент (PCA),
без кластеризации и без визуализации.

Мы анализируем только residual-пространство.

---

# I. Пространство для анализа

Использовать residual-признаки:

- ΔV4_left_residual
- ΔV4_right_residual
- ΔV5_left_residual
- ΔV5_right_residual

Опционально добавить:

- Asym_ΔV1_abs
- Asym_ΔV1_rel
- PSI_tau

Но основной анализ — на residual V4/V5.

---

# II. Подготовка данных

1. Использовать только субъекты,
   прошедшие фильтрацию в production-run.

2. Стандартизировать все признаки (z-score).

3. Проверить отсутствие NaN.

---

# III. PCA (без визуализации)

Выполнить:

1. Расчёт ковариационной матрицы.
2. Расчёт собственных значений.
3. Расчёт доли объяснённой дисперсии.

Вывести:

- Eigenvalues
- % variance explained
- cumulative variance

---

# IV. Формальные критерии

Зафиксировать:

1. Если PC1 ≥ 65% → выраженная одномерность.
2. Если PC1 ≥ 80% → почти полная одномерность.
3. Если первые 2 компоненты ≥ 80% → двумерная структура.
4. Если ≥ 3 компонент существенны (eigenvalue > 1) →
   многомерная структура.

---

# V. Дополнительная проверка

1. Рассчитать:

   - Participation Ratio
   - Effective Rank

2. Рассчитать condition number ковариационной матрицы.

---

# VI. Отчёт

Создать:

Task_27_3E_Latent_Dimensionality_Report.md

Структура:

1. Матрица собственных значений
2. Доля дисперсии
3. Cumulative variance
4. Participation ratio
5. Effective rank
6. Condition number

Без выводов.
Без интерпретации.
Без кластеров.

# VII. Ограничения

Запрещено:

- Делать кластеризацию
- Делать UMAP
- Делать t-SNE
- Делать графики
- Делать заключение A/B/C

# Цель этапа

Определить,
является ли residual-пространство:

- одномерным,
- двумерным,
- или многомерным.

Это финальный математический тест перед ручным выводом.
