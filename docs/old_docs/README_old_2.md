# NeuroTransAnalytics

## Аналитический проект ретроспективного анализа скоростей зрительных реакций (СЗР)

---

## 1. Назначение проекта

Проект **NeuroTransAnalytics** предназначен для
ретроспективного анализа данных тестирования скоростей зрительных реакций (СЗР),
накопленных за многолетний период.

Цель проекта:

- формализовать объект анализа на основе исторических данных;
- провести физиологически и методически корректный анализ СЗР;
- выявить ограничения, неточности и методические слабые места
  ранее применявшейся системы тестирования;
- создать строгую аналитическую основу для интерпретации данных
  **без привлечения дорогостоящих внешних медицинских обследований**.

Проект **не является** проектом разработки новой методики тестирования.
Результаты анализа используются для проверки корректности уже проведённых исследований
и как основание для возможных будущих методических улучшений.

---

## 2. Характер анализируемых данных

Исходные данные представляют собой результаты
комплексного чередующегося тестирования СЗР,
включающего несколько зрительных тестов,
проводимых в рамках одной сессии.

Ключевые особенности данных:

- данные получены в реальных клинических и прикладных условиях;
- тестирование проводилось без выделения моторного компонента;
- отсутствует история повторов отдельных попыток;
- в базе данных присутствуют только финальные значения RT;
- сохраняются счётчики пропусков и преждевременных реакций;
- структура данных фиксирована и не допускает реконструкции хода теста.

---

## 3. Архитектура аналитического проекта

Аналитическая архитектура проекта строго иерархична
и построена по принципу отсечения избыточных степеней свободы.

Последовательность уровней:

Экспериментальный дизайн и метаданные
↓
Исходные данные (boxbase)
↓
ETL → объект анализа (visual_tests)
↓
QC (контроль качества)
↓
Агрегатные расчёты
↓
Анализ вариабельности RT
↓
Сравнительный (relational) анализ



Каждый уровень формализован в отдельном нормативном документе.

## 5. Методологические принципы проекта

В проекте строго соблюдаются следующие положения:

- данные не реконструируются постфактум;
- отсутствует подмена анализа коррекцией данных;
- вариабельность RT рассматривается как информативный показатель;
- агрегирование и сравнение выполняются на строго разведённых уровнях;
- интерпретации не выходят за пределы операционально определённых показателей.

## 8. Итог

**NeuroTransAnalytics** — это не просто анализ данных,
а строго формализованная система ретроспективной проверки
физиологической и методической корректности тестирования СЗР.

Проект предназначен для специалистов,
работающих с когнитивными и сенсомоторными реакциями,
и ориентирован на воспроизводимость, прозрачность и методическую строгость.
