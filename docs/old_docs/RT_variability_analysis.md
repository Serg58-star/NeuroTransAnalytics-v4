# RT_variability_analysis.md

## Аналитический слой оценки вариабельности скоростей зрительных реакций

---

## 1. Назначение документа

Настоящий документ описывает **анализ вариабельности скоростей зрительных реакций (RT)**
как самостоятельный аналитический слой,
выполняемый поверх агрегатов и QC-валидных данных.

Цель анализа вариабельности:

- оценка **стабильности функционирования** зрительной системы;
- выявление **внутрисерийной и межусловной неустойчивости**;
- обнаружение **методических ограничений** тестовой процедуры;
- определение границ интерпретации медианных показателей.

Документ **не рассматривает вариабельность как шум по умолчанию**.

---

## 2. Место анализа вариабельности в архитектуре

Последовательность уровней:

visual_tests
↓
QC
↓
Aggregates (minimal / extended)
↓
RT variability analysis ← (настоящий документ)
↓
Relational analysis


Анализ вариабельности:

- не заменяет агрегаты;
- не модифицирует данные;
- дополняет интерпретацию результатов.

---

## 3. Объект анализа вариабельности

Единица анализа:

> **одна запись `visual_tests`**
> (один тест одного пациента в одной сессии).

Анализ проводится:

- в пределах одного теста;
- без объединения разных тестов;
- без межиндивидуальных сравнений.

---

## 4. Источники данных

Для анализа вариабельности используются:

- `rt_values[]` — финальные значения RT;
- минимальные агрегаты:
  - `median_rt`,
  - `mad_rt`;
- расширенные агрегаты (при наличии):
  - латерализационные,
  - канальные;
- счётчики:
  - `missed_count`,
  - `premature_count`.

QC-статус обязателен к учёту.

---

## 5. Типы вариабельности (принципиальное разделение)

### 5.1. Внутрисерийная вариабельность

Характеризует разброс RT **в пределах одного теста**.

Примеры показателей:

- MAD;
- межквартильный размах;
- диапазон (min–max) с осторожной интерпретацией.

Назначение:
- оценка устойчивости реакции;
- выявление нестабильных режимов выполнения теста.

---

### 5.2. Контекстная вариабельность

Оценивается через:

- сочетание разброса RT;
- количества пропусков и преждевременных реакций;
- изменения распределения RT в пределах теста.

Назначение:
- выявление перегрузки;
- возможного утомления;
- нестабильности внимания.

---

### 5.3. Латерализационная вариабельность

Вариабельность RT **между зонами поля зрения**:

- Left / Center / Right.

Назначение:
- выявление асимметрий устойчивости;
- обнаружение зон с повышенной нестабильностью реакции.

---

## 6. Методологический статус вариабельности

Принципиальные положения:

- высокая вариабельность:
  - **не является автоматически дефектом данных**;
  - **не является автоматически артефактом**;
- вариабельность может отражать:
  - функциональное состояние системы;
  - ограничения методики;
  - индивидуальные особенности реакции.

Запрещается:

- автоматически исключать данные по критерию вариабельности;
- «корректировать» RT для уменьшения разброса;
- подменять анализ вариабельности QC-отбраковкой.

---

## 7. Связь с интерпретацией ретроспективных данных

Анализ вариабельности используется для:

- оценки надёжности медианных показателей;
- выявления серий, требующих осторожной интерпретации;
- объяснения расхождений между тестами;
- анализа ограничений старой методики тестирования.

Он **не предназначен** для:

- клинической диагностики;
- индивидуальной нормы;
- автоматических выводов о патологии.

---

## 8. Роль анализа вариабельности в проекте

В рамках текущего проекта анализ вариабельности RT:

- является **инструментом методической рефлексии**;
- позволяет выявить:
  - где методика стабильна,
  - где она даёт избыточную свободу,
  - где требуется уточнение протокола;
- служит основой для:
  - корректной интерпретации старых данных;
  - проектирования более строгих тестов в будущем.

---

## 9. Статус документа

Настоящий документ:

- является нормативной спецификацией слоя анализа вариабельности RT;
- дополняет агрегатный и relational-анализ;
- не изменяет ранее зафиксированные уровни архитектуры.

Изменения допускаются только при:
- расширении набора регистрируемых параметров;
- изменении структуры `visual_tests`;
- уточнении методики тестирования.

---

## Контрольная фиксация

После утверждения данного документа:

- вариабельность RT рассматривается как самостоятельный аналитический объект;
- подавление разброса без явного обоснования считается методологической ошибкой;
- интерпретация агрегатов всегда сопровождается оценкой устойчивости данных.
