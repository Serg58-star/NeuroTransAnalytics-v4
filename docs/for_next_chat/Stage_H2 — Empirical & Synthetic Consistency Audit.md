# Stage_H2 — Empirical & Synthetic Consistency Audit

## Исполнитель: GoAn

## Роль ChatGPT: последующая независимая методическая проверка

---

# 1. Основание

После Stage 7 (реальные данные), Stage 57–63 (архитектурный кризис) выявлены нерешённые вопросы:

1. Насколько synthetic v5 была структурно изоморфна Stage 7?
2. Сохраняется ли Stage 7 (3D manifold) при разделении по полу?
3. Изменяется ли геометрия Stage 7 при более глубокой возрастной стратификации (спектр, анизотропия, ранг)?
4. Где именно возникло расхождение: в эмпирике или в синтетике?

Данная задача — не пересчёт v5.
Это forensic-аудит соответствия эмпирики и синтетики.

---

# 2. Общие ограничения

Запрещено:

* Использовать κ.
* Использовать Severity.
* Использовать v5 Z-space.
* Вводить демографические поправки.
* Менять архитектуру.
* Делать интерпретационные выводы.

Разрешено:

* Анализировать структуру.
* Сравнивать спектры.
* Сравнивать ковариации.
* Проверять топологию.
* Работать только с эмпирикой Stage 7 и synthetic генератором.

Каждый блок → отдельный .md документ на русском языке.

## Блок 1 — Synthetic Isomorphism Audit

### Цель

Проверить, насколько synthetic данные v5 структурно изоморфны эмпирическому пространству Stage 7.

### Проверить:

1. Размерность (Effective Rank, PR).
2. Спектр собственных значений (λ₁…λ₃).
3. Отношение λ₁ / λ₃.
4. Ковариационную структуру.
5. Плотностную неоднородность (Hopkins).
6. Наличие/отсутствие плотностных градиентов.
7. Топологию (связность, persistent homology — если применимо).

### Deliverable:

`docs/audit/H2_Block1_Synthetic_Isomorphism_Audit.md`

Должен содержать:

* Таблицу «Empirical vs Synthetic».
* Формальное заключение:

  * ISOMORPHIC
  * PARTIALLY_ISOMORPHIC
  * NON_ISOMORPHIC

Без объяснений причин. Только структурное сравнение.

## Блок 2 — Stage 7 Geometry Replication by Sex

### Цель

Проверить, сохраняется ли 3D continuous manifold отдельно у мужчин и женщин.

### Проверить:

1. Hopkins.
2. Gap Statistic.
3. Silhouette peaks.
4. PR / Effective Rank.
5. λ₁ / λ₃.
6. Топологическая связность.

### Deliverable:

`docs/H2_Block2_Sex_Stratified_Geometry.md`

Должен содержать:

* Отдельные таблицы для Male / Female.
* Сравнительный summary.
* Формальное заключение:

  * GEOMETRY_STABLE
  * GEOMETRY_MODIFIED
  * GEOMETRY_COLLAPSED
