# NeuroTransAnalytics Research Control Document (Chat Bootstrap)

## Контрольный документ исследования (обязательный контекст для нового чата)

---

# 1. Назначение документа

Этот документ **обязан размещаться в начале каждого нового аналитического чата проекта NeuroTransAnalytics**.

Его цель — предотвратить потерю методического контекста.

Без понимания:

* архитектуры эксперимента
* структуры стимулов
* алгоритма выполнения теста

анализ базы данных становится бессмысленным.

---

# 2. Что именно исследуется

Проект **не занимается клинической диагностикой**.

Объект исследования:

**взаимодействие детерминированной структуры визуальных стимулов
с нейрофизиологическими свойствами зрительно-моторной системы человека.**

Каждый стимул в тесте является **контролируемым экспериментальным фактором**.

Следовательно:

```
данные = серия повторённых экспериментальных воздействий
```

а не просто набор реакций.

---

# 3. Экспериментальный протокол

Каждое тестирование состоит из **трёх тестов**.

| Тест | Функциональный смысл      | Нейрофизиологическая интерпретация    |
| ---- | ------------------------- | ------------------------------------- |
| Tst1 | простая реакция           | базовый зрительно-моторный канал (V1) |
| Tst2 | реакция на красный сигнал | цветовая обработка (ΔV4)              |
| Tst3 | реакция на движение       | обработка движения (ΔV5 / MT)         |

Каждый тест содержит:

```
36 валидных стимулов
```

перед которыми идут:

```
3 тренировочных
```

которые не записываются.

---

# 4. Алгоритм выполнения теста

Тест выполняется по строгому алгоритму.

Ключевые правила:

1. принимаются только реакции в интервале

```
135 – 2000 мс
```

2. контролируется вариация реакций

```
CV ≤ 15–20 %
```

если новая реакция увеличивает CV выше порога:

```
попытка повторяется
```

Таким образом:

```
в базе находятся уже статистически стабилизированные серии
```

из 36 реакций.

---

# 5. Структура стимулов

Каждый стимул определяется тремя параметрами.

| параметр  | значение                     |
| --------- | ---------------------------- |
| цвет      | красный / зелёный / синий    |
| положение | левое / центральное / правое |
| PSI       | интервал перед стимулом      |

Последовательности стимулов **детерминированы** и одинаковы для всех испытуемых.

Это означает:

```
номер стимула = известное экспериментальное условие
```

---

# 7. Основная идея анализа

Проект исследует **функциональные компоненты зрительной обработки**.

Используются три индекса:

```
V1 = Tst1
ΔV4 = Tst2 − Tst1
ΔV5/MT = Tst3 − Tst1
```

Это позволяет разделять вклад разных каналов обработки.

---

# 8. Уникальность базы данных

Особенности базы:

* одинаковые стимулы для всех испытуемых
* строго фиксированная последовательность
* многолетний сбор данных
* большая выборка (1500+ испытуемых)

Это делает базу **серией повторённых экспериментов**, а не обычной медицинской статистикой.

---

# 9. Главный принцип анализа

Анализ должен рассматривать **стимулы как независимые переменные**.

Обязательно анализировать влияние:

```
позиции
цвета
PSI
номера стимула
```

на время реакции.

---

# 10. Уже выявленные закономерности (Stage L)

В ходе анализа legacy-дизайна обнаружены следующие потенциальные эффекты:

1. асимметрия полей зрения
2. зависимость реакции от PSI
3. эффект переходов между стимулами
4. рост вариабельности в середине серии
5. различия реакций на цвет и движение
6. устойчивая структура вариации RT

Эти эффекты требуют проверки на полной базе.

---

# 11. Правильный порядок анализа

Любой анализ должен выполняться строго по этапам.

### 1 Проверка целостности данных

проверить:

```
36 реакций в каждом тесте
```

---

### 2 Реконструкция параметров стимулов

для каждого стимула определить:

```
цвет
позицию
PSI
```

---

### 3 Базовая статистика

рассчитать:

```
mean
median
MAD
SD
CV
```

---

### 4 Функциональные индексы

рассчитать:

```
ΔV4
ΔV5/MT
```

---

### 5 Анализ влияния стимулов

исследовать зависимости:

```
RT ~ position
RT ~ color
RT ~ PSI
RT ~ stimulus index
```

---

### 6 Популяционные эффекты

исследовать влияние:

```
возраста
пола
повторных тестирований
```

---

# 12. Что категорически запрещено

Нельзя начинать анализ:

* не понимая структуры стимулов
* не зная алгоритма теста
* без связи с экспериментальным дизайном

Иначе получаются **бессмысленные статистические результаты**.
