# 1. Основание

Stage_H3.1 показал:

* F1 baseline историки и v5 структурно несовместимы.
* 1D-доминантность baseline является нейрональной, а не моторной.
* Моторный латентный слой не разрушает коррелированную архитектуру.

Однако v5 проектируется не для воспроизведения «сырых» baseline-времён,
а для анализа **функциональных переходов (Δ)**.

Следовательно, корректное сравнение должно происходить в Δ-пространстве.

---

# 2. Цель Stage_H3.2

Сравнить функциональную геометрию:

* ΔV4 = V4 − V1
* ΔV5 = V5 − V1
* ΔLat (латерализационные переходы)

между:

* исторической эмпирикой (без tapping),
* v5 (baseline-моторно-нормированной архитектурой).

Ключевой вопрос:

> Совпадает ли функциональное Δ-пространство двух систем,
> несмотря на различие baseline?

---

# 3. Формирование Δ-пространства

## 3.1 Историческая выборка

Использовать:

* First-Visit Only (N=1482)
* ΔV4 = tst2 − tst1
* ΔV5 = tst3 − tst1
* ΔLat = (Left − Right) переходы
* Без PSI
* Без моторной симуляции

## 3.2 v5 выборка

Использовать:

* Только Δ-слой
* Отключить:

  * конио
  * Severity
  * κ-порог
* Использовать моторно-нормированную baseline
* Δ рассчитывается аналогично
