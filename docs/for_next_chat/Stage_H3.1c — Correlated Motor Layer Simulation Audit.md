# Stage_H3.1c — Correlated Motor Layer Simulation Audit

## Исполнитель: GoAn

## Роль ChatGPT: независимая методическая проверка

---

# 1. Основание

Stage_H3.1 выявил:

* Эмпирическая F1 Baseline — почти сингулярная 1D структура (Eff.Rank ≈ 1.26).
* Каналы L/C/R сильно коррелированы (r > 0.92).
* Демографическая геометрия выражена амплитудно (Sex λ₁, Q2-коллапс).
* v5 F1 не воспроизводит эту структуру.

Однако исторические данные не содержат теппинг-теста.
Следовательно, baseline включает моторный компонент.

Ключевой вопрос:

> Является ли 1D-доминантность F1 следствием латентного моторного фактора,
> или это нейрональный глобальный baseline?

Для проверки вводится **коррелированный моторный слой**.

---

# 2. Концептуальная модель

Пусть:

[
RT_i = Neural_i + Motor_i
]

Предполагается, что:

[
Corr(Motor_i,\ RT_i) \gg 0
]

Моторный компонент моделируется как линейная функция глобального baseline-фактора.

---

# 3. Формирование глобального baseline-фактора

## 3.1 Вычислить PC1

Для эмпирической F1 (L, C, R):

* Построить ковариационную матрицу.
* Вычислить PC1.
* Получить PC1-score для каждого субъекта:
  [
  G_i
  ]

G_i интерпретируется как глобальный baseline-speed фактор.

Deliverable:

`docs/audit/H3_1c_Block0_Global_Factor.md`

---

# 4. Моделирование коррелированного Motor_i

Мотор моделируется как:

[
Motor_i = \alpha \cdot G_i + \eta_i
]

где:

* α ∈ [0.6, 0.9]
* η_i ~ N(0, σ²), σ = 5–15 мс

Провести Monte Carlo симуляцию:

* 500–1000 итераций
* Для каждого α из диапазона

Deliverable:

`docs/audit/H3_1c_Block1_Motor_Model_Definition.md`

---

# 5. Коррекция baseline

Для каждой симуляции:

[
Neural_i^{est} = RT_i - Motor_i
]

Пересчитать для скорректированного baseline:

* Корреляции L/C/R
* Eff.Rank
* PC1%
* λ₁

# 7. Формальные критерии интерпретации

Если после коррекции:

### A) 1D-доминантность резко ослабляется

(Eff.Rank ↑, PC1% ↓ существенно)

→ BASELINE_MOTOR_DOMINATED

### B) Структура частично ослабляется

→ BASELINE_MIXED (Motor + Neural global factor)

### C) Структура сохраняется

→ BASELINE_NEURAL_GLOBAL
