### 1. Severity формула

`S(Z) = Z^T \Sigma^{-1} Z`

### 2. Anchored Projection формула

`P_anchored(v) = v - (v \cdot u_anchored) * u_anchored` (где `u_anchored` - вектор базовой группы/нормы)

### 3. ΔZ формула

`ΔZ = Z_test - Z_baseline`

### 4. Параметр Generator Stability

`κ = 0.08`

### 5. Anchored thresholds (S75, DII75)

Зафиксированы на значениях базового распределения (при κ=0).
