# Robust Statistics Standard
## NeuroTransAnalytics v4 — Statistical Estimation Rule

Project: **NeuroTransAnalytics v4**  
Scope: **All analytical stages (Stage-L and subsequent analysis layers)**

---

## 1. Назначение

Настоящий документ фиксирует **обязательный стандарт статистических оценок**, используемых в анализе данных проекта NeuroTransAnalytics.

Данный стандарт вводится для обеспечения:

- устойчивости оценок при малых выборках;
- воспроизводимости аналитических результатов;
- исключения вариативности выбора статистических метрик.

---

## 2. Основной принцип

Во всех аналитических процедурах проекта используются **только робастные оценки**.
