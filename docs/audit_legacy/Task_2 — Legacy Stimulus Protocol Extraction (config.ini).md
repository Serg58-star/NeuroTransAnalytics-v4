# 1. Цель задачи

Извлечь полную структуру файла **config.ini**, который используется
в legacy системе тестирования скоростей зрительных реакций.

Файл расположен в каталогах:

Reactest\Тест 1\
Reactest\Тест 2\
Reactest\Тест 3\

Этот файл определяет:

- последовательность стимулов
- параметры теста
- временные интервалы
- конфигурацию эксперимента

На данном этапе требуется **только извлечение структуры**, без интерпретации.

---

# 2. Что необходимо извлечь

## 2.1 Список всех разделов

Например:

[MAINCONFIG]
[T1_CONFIG]
[T2_CONFIG]
[T3_CONFIG]
[mouse]


Для каждого раздела перечислить **все параметры**.

---

## 2.2 Полный список ключей

Для каждого параметра указать:

| Section | Key | Value |

---

## 2.3 Параметры, связанные со стимулами

Отдельно извлечь ключи, содержащие:

- sequence
- arr
- stim
- color
- pos
- disc
- circle

---

## 2.4 Параметры времени

Отдельно извлечь параметры:

- interval
- delay
- rotate
- timer
- period

---

## 2.5 Параметры PSI

Найти параметры, связанные с:

- rotation
- shift
- sequence
- pattern

---

# 3. Формат выходных документов

Создать каталог:

docs/legacy/stimulus_protocol/

Создать следующие документы.

---

## config_sections.md

Список всех разделов config.ini.

---

## config_full_index.md

Полная таблица параметров:

| section | key | value |

---

## config_stimulus_parameters.md

Все параметры, связанные со стимулами.

---

## config_timing_parameters.md

Все параметры времени.

---

## config_sequence_data.md

Если в config.ini присутствуют:

- массивы стимулов
- последовательности

извлечь их **без изменений**.
