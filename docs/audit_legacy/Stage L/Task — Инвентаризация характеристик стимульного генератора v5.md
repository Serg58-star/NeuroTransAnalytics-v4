# Task — Инвентаризация характеристик стимульного генератора v5
## NeuroTransAnalytics v5

Исполнитель: GoAn  
Назначение: выявление всех параметров и алгоритмических характеристик стимульного генератора, уже реализованного в проекте.

---

# 1. Цель задачи

Определить **полный набор характеристик и параметров**, заложенных в стимульный генератор проекта NeuroTransAnalytics v5.

Результат должен позволить:

1. понять реальную архитектуру генератора стимулов;
2. зафиксировать все используемые параметры и ограничения;
3. сопоставить их с эмпирическими результатами Stage-L анализа.

Задача **не предполагает изменения генератора** — требуется только его точная реконструкция и описание.

---

# 2. Источники анализа

Необходимо проанализировать:

1. исходный код генератора стимулов;
2. конфигурационные файлы тестов;
3. файлы описания протокола тестирования;
4. связанные документы проекта.

Особое внимание уделить следующим файлам и разделам:

- config_sequence_data.md
- config_stimulus_parameters.md
- config_timing_parameters.md
- config_sections.md
- config_full_index.md
- LEGACY_RUNTIME_ALGORITHM.md
- Stimulus_Code_Extracts.md
- Delphi_Project_Structure.md

Также проверить возможное наличие генератора в:

- src/
- analysis/
- generator/
- config/
- legacy/

---

# 3. Необходимо выявить

Для стимульного генератора нужно определить:

## 3.1 Структуру теста

- количество стимулов в серии
- количество серий
- структура блоков
- правила перехода между блоками

---

## 3.2 Пространственные параметры

- набор позиций стимулов
- геометрия поля
- правила чередования позиций
- ограничения повторов позиции

---

## 3.3 Цветовые параметры

- набор цветов стимулов
- правила их распределения
- ограничения повторов цвета
- возможные балансировки

---

## 3.4 Временные параметры

- диапазон PSI
- распределение PSI
- шаг дискретизации PSI
- правила генерации PSI последовательности

---

## 3.5 Алгоритм генерации последовательности

Необходимо определить:

- используется ли детерминированная схема
- используется ли случайная генерация
- используется ли псевдослучайная генерация
- используется ли Марковская модель
- существуют ли анти-предсказательные ограничения

---

## 3.6 Ограничения генератора

Выявить правила:

- запрет одинаковых стимулов подряд
- ограничение повторов позиции
- ограничение повторов цвета
- ограничения PSI

---

## 3.7 Параметры случайности

Если используется случайная генерация, определить:

- тип генератора случайных чисел
- правила инициализации
- фиксирован ли seed
- используется ли детерминированная репликация последовательностей

---

## 3.8 Дополнительные механизмы

Проверить наличие:

- маскирования
- фоновых паттернов
- динамических фонов
- адаптивных пауз
- адаптивных повторов
