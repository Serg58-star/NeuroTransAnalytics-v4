Необходимо только:

- перечислить файлы

- сохранить их для последующего анализа.

---

# 2. Что необходимо сделать

## 2.1 Построить индекс файлов проекта

Найти все файлы в указанном каталоге и его подкаталогах.

Особенно важны:

Delphi source files:

- *.pas
- *.dfm
- *.dpr
- *.dpk
- *.res
- *.cfg

Для каждого файла указать:

| Field | Description |
|------|-------------|
| file_name | имя файла |
| path | полный путь |
| type | тип файла |
| size | размер |
| modified_date | дата изменения |

---

## 2.2 Определить структуру Delphi-проекта

Найти основной файл проекта:

*.dpr

Извлечь:

- список подключённых модулей (uses)
- имена форм
- порядок инициализации программы.

---

## 2.3 Определить основные модули системы

По файлам *.pas извлечь:

- имя модуля
- список процедур
- список функций
- список глобальных переменных
- комментарии (если есть).

---

## 2.4 Найти код, связанный со стимулами

Не анализируя логику, найти участки кода,
где используются ключевые слова:

stim
signal
color
draw
canvas
pixel
timer
interval
random
position
x
y
rect
circle

Извлечь **фрагменты кода** вокруг этих участков.

---

## 2.5 Найти код регистрации реакции

Поиск по ключевым словам:

key
mouse
click
reaction
time
keypress
keydown
latency
response

Извлечь соответствующие функции.

---

## 2.6 Найти код обработки результатов

Поиск по ключевым словам:

mean
sd
average
median
variance
result
score
stat
analysis

Извлечь соответствующие участки.
