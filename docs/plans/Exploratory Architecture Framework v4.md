# Exploratory Architecture Framework v4

## Назначение документа

Документ фиксирует полный перечень направлений исследовательских преобразований массива NeuroTransAnalytics-v4.

Цель:

> Выявить, какие преобразования сохраняют латентную геометрию,
> а какие её разрушают.

Тем самым определить границы валидности измерительной архитектуры.

Документ задаёт системную последовательность закрытия направлений I–XXVII.

---

# ЭТАП 1. Инвариантность к масштабированию и метрике

## I. Лог-преобразование

Проверка сохранения размерности при логарифмировании RT.

## II. Box–Cox оптимизация

Поиск λ, при котором пространство максимально гауссово.

## III. Rank-based PCA

Проверка сохранения структуры при использовании рангов.

## IV. Z-нормировка внутри субъекта

Отделение относительной структуры от абсолютной скорости.

## V. Нормировка внутри теста

Проверка метрической устойчивости между тестами.

**Цель этапа:**
Определить, является ли структура метрической или порядковой.

---

# ЭТАП 2. Координатные преобразования

## VI. Преобразование латерализации

(L + R)/2 — общая скорость
(L − R) — асимметрия

## VII. Центр vs Периферия

Отдельная геометрия для центрального и периферического поля.

## VIII. Полярные координаты латентного пространства

Радиус (общая выраженность)
Угол (профиль V4–V5 баланса)

## IX. PCA только по асимметриям

Анализ пространства (Left − Right) как самостоятельной геометрии.

**Цель этапа:**
Выявить независимую ось асимметрии и проверить роль латерализации.

# ЭТАП 3. Канальная реконструкция

## X. Parvo / Magno индексы

Re-aggregation по предполагаемым зрительным каналам.

## XI. Koniocellular компонент

Выделение синего канала как отдельного фактора.

## XII. Корреляционная матрица каналов

Corr(Parvo, Magno) между субъектами и внутри субъекта.

## XIII. Канальные профили субъектов

Выявление доминирующего канала.

**Цель этапа:**
Проверка нейрофизиологической валидности осей.

# ЭТАП 4. Стимул-конструктивная декомпозиция

## XIV. PSI-стратификация

Отдельная латентная структура для разных PSI.

## XV. Нелинейность PSI

Экспоненциальная / сигмоидальная / пороговая модель.

## XVI. Производная ΔRT/ΔPSI

Анализ чувствительности к интервалу.

## XVII. Shift_parameter (0–3)

Отдельная геометрия для статичных и динамических стимулов.

## XVIII. Параметризация mask_triples

Оценка сложности маски (энтропия, число переходов, симметрия).

## XIX. Response-to-structure индекс

**Цель этапа:**
Определить вклад конструктивных свойств стимула в формирование осей.

# Стратегическая логика порядка этапов

1. Сначала — минимальные преобразования (масштаб).
2. Затем — координатные.
3. Потом — канальные.
4. Далее — конструктивные свойства стимула.
5. Затем — микродинамика.
6. Далее — вариативность и хвосты.
7. Потом — нелинейная геометрия.
8. Затем — продольная динамика.
9. В конце — стресс-тест разрушения.

Такой порядок позволяет двигаться:

от минимального вмешательства
к максимально инвазивным преобразованиям,

сохраняя интерпретируемость каждого шага.
